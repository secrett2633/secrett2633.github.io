<h1>PEP 8103 – 2022년도 Steering Council 선거</h1>
<ul>
<li><strong>작성자:</strong> Ewa Jodlowska, Ee Durbin, Joe Carey</li>
<li><strong>후원자:</strong> Barry Warsaw</li>
<li><strong>상태:</strong> Final</li>
<li><strong>유형:</strong> Informational (정보성)</li>
<li><strong>주제:</strong> Governance (거버넌스)</li>
<li><strong>생성일:</strong> 2021년 10월 4일</li>
</ul>
<h2>개요 (Abstract)</h2>
<p>이 문서는 PEP 13에 명시된 대로 2021년 12월에 진행된 Python Steering Council (운영 위원회) 선거의 일정 및 기타 세부 사항을 설명합니다. 이는 2022년 임기(즉, Python 3.11 개발 기간)를 위한 Steering Council 선거입니다.</p>
<h2>선거 관리 (Election Administration)</h2>
<p>Steering Council은 Python Software Foundation의 인프라 디렉터인 Ee Durbin과 회계 관리자인 Joe Carey를 선거 공동 관리자로 임명했습니다.</p>
<h2>일정 (Schedule)</h2>
<p>지명 기간(nomination period)은 2주, 투표 기간(voting period)은 2주로 진행되었습니다.</p>
<ul>
<li><strong>지명 기간:</strong> 2021년 11월 1일부터 2021년 11월 16일 12:00 UTC (Anywhere on Earth 기준 2021년 11월 15일 말)까지였습니다. 이 내용은 <code>python-committers</code> 메일링 리스트에 공지되었고, <code>discuss.python.org</code>에서 진행되었습니다.</li>
<li><strong>투표 기간:</strong> 2021년 12월 1일 12:00 UTC부터 2021년 12월 16일 12:00 UTC (Anywhere on Earth 기준 2021년 12월 15일 말)까지였습니다.</li>
</ul>
<h2>후보자 (Candidates)</h2>
<p>후보자는 코어 팀 멤버에 의해 지명되어야 합니다. 후보자가 코어 팀 멤버인 경우 스스로를 지명할 수 있습니다.</p>
<p>지명된 후보자(알파벳 순):</p>

</ul>
<p>철회된 지명은 없었습니다.</p>
<h2>유권자 명부 (Voter Roll)</h2>
<p>모든 활동 중인 Python 코어 팀 멤버는 투표 자격이 있습니다. 활동 상태는 PEP 13에 설명된 대로 결정되며, <code>python/voters</code> 소프트웨어를 통해 구현됩니다.</p>
<p>투표 용지는 이 선거를 위한 Python 유권자 명부(<code>The Python Voter Roll</code>)를 기반으로 배포되었습니다.
이 파일은 개인 이메일 주소를 포함하므로 공개되지 않지만, 명부가 생성되면 이름으로 된 전체 유권자 명부는 공개될 예정이었습니다.</p>
<h2>선거 구현 (Election Implementation)</h2>
<p>선거는 Helios Voting Service를 사용하여 진행되었습니다.</p>
