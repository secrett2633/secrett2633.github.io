<h3><strong>제목</strong></h3>
<p>PEP 373 – Python 2.7 릴리스 스케줄</p>
<h3><strong>작성자 및 상태</strong></h3>
<ul>
<li><strong>작성자:</strong> Benjamin Peterson </li>
<li><strong>상태:</strong> Final (최종)</li>
<li><strong>유형:</strong> Informational (정보성)</li>
<li><strong>주제:</strong> Release (릴리스)</li>
<li><strong>생성일:</strong> 2008년 11월 3일</li>
<li><strong>Python 버전:</strong> 2.7</li>
</ul>
<h3><strong>개요 (Abstract)</strong></h3>
<p>이 문서는 Python 2.7의 개발 및 릴리스 스케줄을 설명합니다. Python 2.7은 Python 2.x 시리즈의 마지막 버전이며, Python 3으로 계승됩니다. 일반적인 개요는 python.org의 "Sunsetting Python 2" FAQ를 참조하십시오.</p>
<h3><strong>2014년 4월 업데이트 (Update (April 2014))</strong></h3>
<p>Python 2.7의 EOL (End Of Life, 지원 종료일) 또는 sunset date가 5년 연장되어 2020년으로 변경되었습니다. 이 결정은 Python 2.7의 상태를 명확히 하고 아직 Python 3으로 마이그레이션할 수 없는 사용자들의 우려를 해소하기 위해 이루어졌습니다. PEP 466도 참고하십시오.</p>
<p>이 선언은 버그 수정 릴리스가 정기적으로 이루어질 것을 보장하지는 않지만, Python 2.7에 버그 수정을 기여하고자 하는 자원봉사자들을 지원하고 앞으로도 수년간 Python 2를 지원해야 하는 벤더(vendor)들의 요구를 충족시킬 것입니다.</p>
<p>Python 2.8은 존재하지 않을 것입니다 (PEP 404 참조).</p>
