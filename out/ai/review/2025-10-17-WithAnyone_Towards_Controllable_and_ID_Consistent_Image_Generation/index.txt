<p><strong>저자:</strong> Hengyuan Xu, Wei Cheng, Peng Xing, Yixiao Fang, Shuhan Wu, Rui Wang, Xianfang Zeng, Daxin Jiang, Gang Yu, Xingjun Ma, Yu-Gang Jiang</p>
<h2>핵심 연구 목표</h2>
<p>본 논문은 텍스트-투-이미지 생성 모델에서 레퍼런스 인물의 ID(Identity)를 일관성 있게 유지하면서도, 레퍼런스 이미지를 단순히 복사하는 듯한 <strong>"copy-paste" 아티팩트</strong>를 줄이고 생성된 이미지의 <strong>표현, 포즈, 조명 등의 다양성 및 제어 가능성</strong>을 높이는 것을 목표로 합니다. 기존 모델들이 과도한 ID 유사성을 추구하여 발생하는 제어 가능성 저하 문제를 해결하고자 합니다.</p>
<h2>핵심 방법론</h2>
<p>연구팀은 첫째, 다중 인물 시나리오를 위한 대규모 페어링 데이터셋인 <strong>MultiID-2M</strong>을 구축하여 각 ID에 대해 다양한 레퍼런스를 제공합니다. 둘째, <strong>MultiID-Bench</strong> 벤치마크를 도입하여 copy-paste 아티팩트와 ID 충실도 및 다양성 간의 트레이드오프를 정량화합니다. 셋째, <strong>FLUX</strong> 아키텍처 기반의 <strong>WithAnyone</strong> 모델에 **ID 대조 손실(contrastive identity loss)**과 <strong>GT-aligned ID 손실</strong>을 포함하는 새로운 훈련 패러다임을 제안하여 충실도와 다양성 간의 균형을 맞춥니다.</p>
<h2>주요 결과</h2>
<p><strong>WithAnyone</strong>은 <strong>MultiID-Bench</strong>에서 copy-paste 아티팩트를 <strong>크게 줄이면서도</strong> 높은 ID 유사성(Sim(GT))을 유지함을 입증했습니다. 예를 들어, 단일 인물 서브셋에서 WithAnyone은 Sim(GT) <strong>0.460</strong>, CP <strong>0.144</strong>를 달성하여, InstantID(Sim(GT) <strong>0.464</strong>, CP <strong>0.337</strong>)보다 copy-paste가 훨씬 적고 ID 충실도는 유사합니다. 또한, 포즈 및 표정에 대한 제어 가능성이 향상되고 전반적인 시각적 품질이 높게 평가되었습니다. 사용자 연구에서도 높은 ID 충실도와 표현력 있는 제어 가능 생성을 확인했습니다.</p>
<h2>AI 실무자를 위한 시사점</h2>
<p><strong>MultiID-2M</strong> 데이터셋과 <strong>MultiID-Bench</strong> 벤치마크는 다중 인물 ID 보존 생성 모델 연구를 위한 귀중한 자원이 될 것입니다. **대조 학습(contrastive learning)**을 활용하여 단순히 레퍼런스를 복제하는 것을 넘어 ID의 본질을 학습하고 제어 가능성을 확보하는 방법론은 실제 애플리케이션에서 <strong>더욱 유연하고 자연스러운 이미지 생성</strong>을 가능하게 할 것입니다. 이는 아바타 생성, 콘텐츠 제작 등 다양한 분야에서 활용될 수 있습니다.</p>
