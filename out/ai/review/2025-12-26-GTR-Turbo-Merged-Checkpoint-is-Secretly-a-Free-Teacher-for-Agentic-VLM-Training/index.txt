<p><strong>저자:</strong> Tong Wei, Yijun Yang, Changhao Zhang, Junliang Xing, Yuanchun Shi, Zongqing Lu, Deheng Ye</p>
<h2>핵심 연구 목표</h2>
<p>멀티턴 강화 학습(RL) 기반 <strong>VLM(Vision-Language Model) 에이전트 훈련</strong> 의 주요 문제점인 희소한 보상, 긴 신용 할당 문제, 그리고 <strong>GTR(Guided Thought Reinforcement)</strong> 과 같은 기존 방법론에서 외부 교사 모델 사용으로 인한 높은 비용과 접근성 한계를 해결하는 것을 목표로 합니다. 외부 교사 모델 없이도 성능을 유지하거나 향상시키는 <strong>효율적이고 확장 가능한(scalable) RL 프레임워크</strong> 를 제안합니다.</p>
<h2>핵심 방법론</h2>
<p><strong>GTR-Turbo</strong> 는 학습 과정에서 생성된 <strong>역사적 체크포인트들의 가중치를 병합</strong> 하여 "무료" 교사 모델을 생성합니다. 매 <strong>PPO(Proximal Policy Optimization) 업데이트</strong> 후 모델 가중치를 저장하고 <strong>TIES merging technique (56)</strong> 을 사용하여 이전 경험들을 통합합니다. 이 병합된 모델은 <strong>SFT(Supervised Fine-tuning) 기반 온라인 모방 학습</strong> 또는 <strong>소프트 로짓 증류(soft logit distillation) (역방향 KL 발산 최소화)</strong> 를 통해 후속 RL 학습을 안내합니다.</p>
<h2>주요 결과</h2>
<p><strong>Points24 카드 게임 태스크</strong> 에서 <strong>GTR-Turbo (KL)</strong> 는 <strong>53.5%의 성공률(SR)</strong> 과 <strong>2.39의 에피소드 보상(ER)</strong> 을 달성하여 기존 <strong>GTR (44.5% SR, 0.53 ER)</strong> 을 크게 능가했습니다. <strong>ALFWorld 환경</strong> 에서는 <strong>GTR-Turbo (KL)</strong> 가 <strong>15% SR</strong> 로 GTR (16% SR)과 유사한 성능을 보였으나, 훈련 시간은 <strong>50% 감소</strong> , 컴퓨팅 비용은 <strong>60% 절감</strong> 되었습니다. 전반적으로 <strong>GTR-Turbo</strong> 는 외부 API 호출 없이도 기준 모델의 정확도를 <strong>10-30% 향상</strong> 시켰습니다.</p>
<h2>AI 실무자를 위한 시사점</h2>
<p><strong>GTR-Turbo</strong> 는 외부의 값비싼 API 기반 교사 모델에 의존하지 않고도 효과적인 <strong>생성형 VLM 에이전트 훈련</strong> 을 가능하게 하는 <strong>실용적인 대안</strong> 을 제공합니다. 학습 중 생성되는 체크포인트를 활용하는 <strong>모델 병합 전략</strong> 은 추가 비용 없이 강력한 교사 신호를 제공함으로써 <strong>훈련 시간과 컴퓨팅 자원을 대폭 절감</strong> 합니다. 특히 <strong>KL 발산 기반의 로짓 증류 방식</strong> 은 효율성과 안정성을 높여, 자원 제약이 있는 환경에서 <strong>멀티턴 시각 에이전트 훈련</strong> 의 접근성과 확장성을 크게 개선할 수 있습니다.</p>
