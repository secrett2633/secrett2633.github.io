<p><strong>저자:</strong> Hongze Mi, Yibo Feng, Wenjie Lu, Yuqi Wang, Jinyuan Li</p>
<h2>핵심 연구 목표</h2>
<p>본 논문은 기존 GUI 에이전트의 데이터 병목 현상, 지연된 오류 탐지의 높은 비용, 모순된 지침 등의 문제점을 해결하고자 합니다. 인간의 인지 루프(사고, 정렬, 성찰)에서 영감을 받은 <strong>D-Artemis</strong>라는 새로운 심의적 인지 프레임워크를 제시하여 모바일 GUI 멀티 에이전트의 견고하고 적응적인 자율 작업을 가능하게 하는 것을 목표로 합니다.</p>
<h2>핵심 방법론</h2>
<p><strong>D-Artemis</strong>는 <strong>세분화된 앱별 팁 검색 메커니즘</strong>을 통해 의사결정 과정을 보강합니다. 또한, 실행 실패 위험을 완화하기 위해 <strong>Thought-Action Consistency (TAC) Check 모듈</strong>과 **Action Correction Agent (ACA)**로 구성된 <strong>선제적 사전 실행 정렬(Pre-execution Alignment)</strong> 단계를 활용하며, <strong>TAC 모듈</strong>은 <strong>Qwen2.5-VL-7B</strong>를 <strong>SFT 튜닝</strong>하여 구축되었습니다. 마지막으로 **사후 실행 Status Reflection Agent (SRA)**가 경험을 통한 전략적 학습을 가능하게 하여 인지 루프를 완성합니다.</p>
<h2>주요 결과</h2>
<p><strong>AndroidWorld 벤치마크</strong>에서 **75.8%**의 성공률을 달성하며 기존 SOTA 모델을 <strong>2.5%p</strong> 능가했습니다. <strong>Qwen2.5-VL-72B-Instruct</strong> 기반 모델 중에서는 **68.1%**로 <strong>UI-Venus</strong>보다 <strong>2.2%p</strong> 높은 성능을 보였습니다. <strong>ScreenSpot-V2 벤치마크</strong>에서는 **96.8%**의 평균 성공률을 기록했으며, 특히 'Icon/Widget' 태스크에서 **95.6%**를 달성하여 강력한 일반화 능력을 입증했습니다.</p>
<h2>AI 실무자를 위한 시사점</h2>
<p>본 프레임워크는 복잡한 궤적 데이터셋 훈련 없이도 일반 목적의 <strong>MLLM</strong>이 GUI 태스크에서 뛰어난 성능을 발휘하도록 하여 <strong>데이터 효율적인 자율 GUI 에이전트</strong> 개발에 중요한 방향을 제시합니다. <strong>선제적 사전 실행 정렬</strong>은 실행 전 오류를 감지하고 수정하여 비용이 많이 드는 태스크 이탈을 방지하는 실용적인 방법론을 제공합니다. 또한, <strong>세분화된 앱별 팁 검색</strong>은 일반적인 지침에서 발생할 수 있는 논리적 충돌을 해결하며 에이전트의 의사결정 능력을 효과적으로 향상시킵니다.</p>
