<h2 id="핵심-연구-목표"><a href="#핵심-연구-목표">핵심 연구 목표</a></h2>
<p>본 논문은 기존 LLM 에이전트 프레임워크의 고질적인 문제인 <strong>비효율적인 순차적 처리 방식</strong> 을 해결하여, 복잡한 웹 기반 추론 작업에서 발생하는 과도한 실행 단계와 긴 지연 시간을 단축하는 것을 목표로 합니다. 특히 광범위한 도구 상호작용이 필요한 태스크에서 에이전트의 효율성과 확장성을 근본적으로 개선하고자 합니다.</p>
<h2 id="핵심-방법론"><a href="#핵심-방법론">핵심 방법론</a></h2>
<p>제안하는 <strong>FLASH-SEARCHER</strong> 는 복잡한 태스크를 명시적 의존성을 가진 하위 태스크로 분해하여 <strong>DAG(Directed Acyclic Graph) 기반 계획</strong> 을 수립합니다. 이 프레임워크는 <strong>동시 실행</strong> 을 통해 독립적인 추론 경로를 병렬로 처리하며, <strong>동적 워크플로우 최적화</strong> 와 요약 모듈을 통합하여 실행 그래프를 지속적으로 개선하고 중복 상호작용을 줄입니다. 도구는 <strong>Serper API 기반 검색 도구</strong> 와 <strong>Jina Reader 기반 크롤링 도구</strong> 만을 사용하여 간결성을 유지하며, <strong>3354개의 DAG 기반 추론 궤적</strong> 으로 구성된 데이터셋을 통해 <strong>지도 미세 조정을 수행</strong> 하여 에이전트를 학습시켰습니다.</p>
<h2 id="주요-결과"><a href="#주요-결과">주요 결과</a></h2>
<p>FLASH-SEARCHER는 <strong>BrowseComp 벤치마크에서 67.7% 정확도</strong> 와 <strong>xbench-DeepSearch에서 83% 정확도</strong> 를 달성하며 기존 최첨단 방식을 뛰어넘는 성능을 보였습니다. 또한, 기존 프레임워크 대비 에이전트 실행 단계를 <strong>최대 35% 감소</strong> 시키고 전체 실행 시간을 <strong>약 65% 단축</strong> 하는 등 현저한 효율성 개선을 입증했습니다. 경량화된 <strong>Qwen-2.5 오픈 소스 모델</strong> 에 적용했을 때도 <strong>xbench-DeepSearch에서 68.0% 성능</strong> 을 기록하며 방법론의 일반화 가능성을 확인했습니다.</p>
<h2 id="ai-실무자를-위한-시사점"><a href="#ai-실무자를-위한-시사점">AI 실무자를 위한 시사점</a></h2>
<p>이 연구는 LLM 에이전트 시스템의 <strong>확장성과 효율성을 혁신</strong> 할 수 있는 <strong>병렬 추론 패러다임</strong> 을 제시합니다. AI 엔지니어는 <strong>DAG 기반 태스크 분해 및 병렬 실행</strong> 을 통해 복잡한 다단계 작업을 훨씬 빠르게 처리하고, 특히 웹 스케일 정보 검색 및 도구 사용이 많은 애플리케이션에서 <strong>운영 비용을 절감</strong> 할 수 있습니다. 또한, 경량 미세 조정만으로 병렬 추론 능력을 다양한 LLM 백본에 전이할 수 있음을 보여주어, 실제 서비스에 적용하기 위한 <strong>모델 배포의 유연성</strong> 을 높였습니다.</p>
