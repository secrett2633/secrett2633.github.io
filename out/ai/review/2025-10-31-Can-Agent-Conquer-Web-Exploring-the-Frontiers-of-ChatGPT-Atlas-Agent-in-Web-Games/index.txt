<p><strong>저자:</strong> Jingran Zhang, Ning Li, Justin Cui</p>
<h2>핵심 연구 목표</h2>
<p>논문은 OpenAI의 <strong>ChatGPT Atlas 에이전트</strong> 가 웹 환경에서 상호작용하는 능력을, 특히 웹 기반 게임을 통해 평가하는 것을 목표로 합니다. 기존 연구가 정보 검색과 같은 정적인 작업에 초점을 맞춘 반면, 본 연구는 <strong>분석적 추론, 입력 실행, 적응 행동, 맥락 이해</strong> 와 같은 다이나믹하고 상호작용적인 환경에서의 에이전트 성능을 심층적으로 탐구하고자 합니다.</p>
<h2>핵심 방법론</h2>
<p>연구는 <strong>ChatGPT Atlas 브라우저</strong> 의 "Agent Mode (Preview)"를 활용하여 <strong>제로샷(zero-shot) 평가 프로토콜</strong> 을 따랐습니다. <strong>Google T-Rex Runner, Sudoku, Flappy Bird, 2048, Stein.world</strong> 등 다양한 유형의 웹 게임을 테스트 시나리오로 선정했으며, 각 게임에서 얻은 <strong>정량적 성능 지표</strong> 와 에이전트의 <strong>질적 행동 분석</strong> 을 결합하여 Atlas의 웹 상호작용 능력을 다각도로 평가했습니다.</p>
<h2>주요 결과</h2>
<p><strong>Sudoku</strong> 와 같은 논리 퍼즐 게임에서는 평균 <strong>2분 28초</strong> 의 완료 시간으로 인간 기준선보다 <strong>4.5배 빠른 속도</strong> 와 100%의 정확도를 달성하며 뛰어난 분석적 추론 능력을 보였습니다. 그러나 <strong>T-Rex Runner</strong> 에서는 인간 기준선 대비 <strong>11.7%</strong> 의 평균 점수를 기록했고, <strong>Flappy Bird</strong> 에서는 10회 시도 모두 <strong>0점</strong> 을 기록하며 실시간 타이밍과 정밀한 조작이 필요한 게임에서 심각한 한계를 드러냈습니다. <strong>2048</strong> 에서는 <strong>64-타일</strong> 수준에서 정체되어 전략적 계획 능력의 부족을 시사했습니다.</p>
<h2>AI 실무자를 위한 시사점</h2>
<p><strong>ChatGPT Atlas</strong> 는 규칙 기반의 분석적 추론 작업에서는 탁월한 성능을 보이지만, 정밀한 모터 제어, 실시간 적응, 내러티브 맥락 이해가 필요한 동적 웹 환경에서는 상당한 한계를 가지고 있습니다. 이는 웹 에이전트 개발 시 <strong>작업의 동적 특성</strong> 을 고려하여, 특히 실시간 상호작용 및 복잡한 전략이 요구되는 애플리케이션의 경우 <strong>모터 제어 및 전략 계획 능력</strong> 에 대한 추가적인 연구 및 개선이 필요함을 강조합니다.</p>
