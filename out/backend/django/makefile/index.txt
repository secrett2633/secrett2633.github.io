9:T770,<p>Django 프로젝트에서 Makefile을 사용하면 개발 과정을 더욱 효율적으로 관리할 수 있습니다.</p>
<h2>Makefile이란?</h2>
<p>Makefile은 소프트웨어 빌드 자동화 도구로, 복잡한 명령어들을 간단한 키워드로 실행할 수 있게 해줍니다.</p>
<h2>Django에서 Makefile 사용하기</h2>
<h3>기본적인 Makefile 예제</h3>
<pre><code class="language-makefile">.PHONY: help install run test migrate

help: ## 도움말 표시
	@echo "사용 가능한 명령어:"
	@grep -E '^[a-zA-Z_-]+:.*?## .*$$' $(MAKEFILE_LIST) | sort | awk 'BEGIN {FS = ":.*?## "}; {printf "\033[36m%-20s\033[0m %s\n", $$1, $$2}'

install: ## 의존성 설치
	pip install -r requirements.txt

run: ## 개발 서버 실행
	python manage.py runserver

test: ## 테스트 실행
	python manage.py test

migrate: ## 데이터베이스 마이그레이션
	python manage.py migrate

makemigrations: ## 마이그레이션 파일 생성
	python manage.py makemigrations

shell: ## Django shell 실행
	python manage.py shell

collectstatic: ## 정적 파일 수집
	python manage.py collectstatic --noinput
</code></pre>
<h3>사용법</h3>
<pre><code class="language-bash">make help          # 도움말 보기
make install       # 의존성 설치
make run          # 개발 서버 실행
make test         # 테스트 실행
make migrate      # 마이그레이션 실행
</code></pre>
<h2>장점</h2>
<ol>
<li><strong>명령어 단순화</strong>: 복잡한 명령어를 간단한 키워드로 실행</li>
<li><strong>팀 협업</strong>: 모든 팀원이 동일한 명령어 사용</li>
<li><strong>문서화</strong>: Makefile 자체가 프로젝트의 사용법을 문서화</li>
<li><strong>자동화</strong>: 반복적인 작업을 자동화</li>
</ol>
