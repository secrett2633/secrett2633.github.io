a:Tf55,<h1 id="들어가며"><a href="#들어가며">들어가며</a></h1>
<p>지난 글에서는 뇌과학적으로 게으름과 무기력을 극복하는 방법에 대해 다루었습니다. 그 방법 중 하나로 <strong>불필요한 정보를 차단하는 방법</strong> 이 있었는데요. 저의 경우 YouTube 시청 시간이 길었기에, YouTube에 접속할 때마다 제가 작성한 뇌과학적으로 게으름과 무기력을 극복하는 글로 <strong>자동 리다이렉트</strong> 해주는 확장 프로그램을 만들었습니다.
이 글에서는 해당 크롬 확장 프로그램을 만드는 방법을 공유하겠습니다.</p>
<h2 id="확장-프로그램-개발"><a href="#확장-프로그램-개발">확장 프로그램 개발</a></h2>
<p>크롬 확장 프로그램을 만들기 위해서는 <strong><code>manifest.json</code> 파일</strong> 과 <strong>백그라운드 스크립트 (<code>background.js</code>)</strong> 가 필요합니다.</p>
<h3 id="1-manifestjson-작성"><a href="#1-manifestjson-작성">1. <code>manifest.json</code> 작성</a></h3>
<pre><code class="language-json">{
    "manifest_version": 3,
    "name": "YouTube Redirector",
    "version": "1.0",
    "description": "YouTube 접속 시 특정 사이트를 새 창에 엽니다",
    "permissions": ["tabs", "webNavigation"],
    "host_permissions": ["*://youtube.com/*"],
    "background": {
      "service_worker": "background.js"
    }
}
</code></pre>
<h4 id="설명"><a href="#설명">설명</a></h4>
<ul>
<li><code>manifest_version: 3</code>: 최신 크롬 확장 프로그램 버전</li>
<li><code>name</code>, <code>version</code>, <code>description</code>: 확장 프로그램의 기본 정보</li>
<li><code>permissions</code>: 확장 프로그램이 사용할 권한 (탭과 웹 탐색 감지)</li>
<li><code>host_permissions</code>: YouTube 사이트에 대한 접근 권한</li>
<li><code>background</code>: 백그라운드 스크립트로 <code>background.js</code> 사용</li>
</ul>
<h3 id="2-backgroundjs-작성"><a href="#2-backgroundjs-작성">2. <code>background.js</code> 작성</a></h3>
<pre><code class="language-js">chrome.webNavigation.onCompleted.addListener(
    function(details) {
      // YouTube 도메인에 접속했을 때만 실행
      if (details.url.includes("www.youtube.com") &#x26;&#x26; details.frameId === 0) {
        // 현재 탭에서 원하는 URL로 이동
        chrome.tabs.update(details.tabId, {
          url: "https://blog.secrett2633.cloud/me/brain-hack-productivity/"
        });
      }
    },
    { url: [{ hostContains: "youtube.com" }] }
);
</code></pre>
<h4 id="설명"><a href="#설명">설명</a></h4>
<ul>
<li><code>chrome.webNavigation.onCompleted.addListener()</code>: 사용자가 웹 탐색을 완료했을 때 실행됨</li>
<li><code>details.url.includes("www.youtube.com")</code>: YouTube 사이트에 접속했는지 확인</li>
<li><code>chrome.tabs.update()</code>: 현재 탭을 원하는 URL로 변경</li>
</ul>
<h2 id="확장-프로그램-설치-방법"><a href="#확장-프로그램-설치-방법">확장 프로그램 설치 방법</a></h2>
<ol>
<li>위 코드를 <code>manifest.json</code>과 <code>background.js</code> 파일로 저장합니다.</li>
<li>크롬에서 <code>chrome://extensions/</code> 페이지로 이동합니다.</li>
<li><strong>개발자 모드</strong> 를 활성화합니다.</li>
<li>"압축해제된 확장 프로그램 로드" 버튼을 클릭하고 해당 폴더를 선택합니다.</li>
<li>확장 프로그램이 정상적으로 등록되면 YouTube 접속 시 자동으로 설정한 URL로 이동됩니다.</li>
</ol>

<p>현재는 로컬에서만 사용할 수 있으며, 실제로 배포하려면 <strong>크롬 웹 스토어에 업로드</strong> 해야 합니다.
