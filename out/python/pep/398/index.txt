<h1>PEP 398 – Python 3.3 릴리스 스케줄</h1>
<ul>
<li><strong>작성자:</strong> Georg Brandl </li>
<li><strong>상태:</strong> Final (최종)</li>
<li><strong>유형:</strong> Informational (정보성)</li>
<li><strong>주제:</strong> Release (릴리스)</li>
<li><strong>생성일:</strong> 2011년 3월 23일</li>
<li><strong>Python 버전:</strong> 3.3</li>
</ul>
<h2>초록 (Abstract)</h2>
<p>이 문서는 Python 3.3의 개발 및 릴리스 스케줄에 대해 설명합니다. 스케줄은 주로 PEP 규모의 항목들을 다룹니다.</p>
<h2>릴리스 관리자 (Release Manager) 및 팀</h2>
<ul>
<li><strong>3.3 릴리스 관리자:</strong> Georg Brandl, Ned Deily (3.3.7 이상)</li>
<li><strong>Windows 인스톨러:</strong> Martin v. Löwis</li>
<li><strong>Mac 인스톨러:</strong> Ronald Oussoren/Ned Deily</li>
<li><strong>문서화 (Documentation):</strong> Georg Brandl</li>
</ul>
<h2>3.3 수명 주기 (Lifespan)</h2>
<p>Python 3.3은 약 18개월 동안 4~6개월마다 버그 수정 업데이트를 받을 예정이었습니다. 3.4.0 final 버전이 릴리스된 후, 최종 3.3 버그 수정 업데이트가 릴리스되었습니다. 그 후, 3.3 final 릴리스 후 5년이 되는 2017년 9월까지 보안 업데이트(소스 코드만 해당)가 릴리스되었습니다.</p>
<p>2017년 9월 29일부로 Python 3.3.x는 EOL (End-Of-Life, 서비스 종료) 상태에 도달했습니다.</p>

<li><a href="https://peps.python.org/pep-3155/">PEP 3155: Qualified name for classes and functions</a> (클래스 및 함수의 한정자 이름)</li>
</ul>
