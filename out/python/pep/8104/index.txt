<p><strong>원문 링크:</strong> <a href="https://peps.python.org/pep-8104/">PEP 8104 - 2023 Term Steering Council election</a></p>
<p><strong>상태:</strong> Final | <strong>유형:</strong> Informational | <strong>작성일:</strong> 08-Nov-2022</p>
</blockquote>
<p>PEP 8104는 "2023 Term Steering Council election"에 대한 정보성(Informational) PEP로, 2022년 12월에 진행된 Python 스티어링 카운슬(Steering Council) 선거의 일정 및 세부 사항을 다루고 있습니다. 이 문서는 Python 3.12 버전에 해당하는 2023년 임기의 스티어링 카운슬 선거에 대한 내용을 명시합니다.</p>
<h2>개요 (Abstract)</h2>
<p>이 문서는 PEP 13에 명시된 바와 같이, Python 스티어링 카운슬의 2022년 12월 선거 일정과 기타 세부 사항을 설명합니다. 이는 2023년 임기(즉, Python 3.12) 스티어링 카운슬 선거입니다.</p>
<h2>선거 관리 (Election Administration)</h2>
<p>스티어링 카운슬은 Python Software Foundation의 인프라 책임자인 Ee Durbin을 선거 관리자로 임명했습니다.</p>
<h2>일정 (Schedule)</h2>
<p>선거는 2주간의 후보 지명 기간과 이어진 2주간의 투표 기간으로 구성되었습니다.</p>
<ul>
<li><strong>후보 지명 기간:</strong> 2022년 11월 14일부터 2022년 11월 28일 AoE(Anywhere on Earth)까지.</li>
<li><strong>투표 기간:</strong> 2022년 12월 1일부터 2022년 12월 14일 AoE(Anywhere on Earth)까지.</li>
</ul>
<h2>후보자 (Candidates)</h2>
<p>후보자는 코어 팀(core team) 멤버에 의해 지명되어야 합니다. 후보자가 코어 팀 멤버인 경우 스스로를 지명할 수 있습니다.</p>
<p><strong>지명된 후보자 (알파벳순):</strong></p>
<ul>
<li>Brett Cannon</li>
<li>Emily Morehouse</li>
<li>Dong-hee Na</li>
<li>Pablo Galindo Salgado</li>
<li>Gregory P. Smith</li>

<li>Petr Viktorin</li>
<li>Thomas Wouters</li>
</ul>
<p><strong>철회된 지명:</strong> 없음</p>
<h2>유권자 명부 (Voter Roll)</h2>
<p>모든 활동 중인 Python 코어 팀 멤버는 투표 자격이 있습니다. 활동 상태는 PEP 13에 설명된 대로 결정되며, <code>python/voters</code> 소프트웨어를 통해 구현됩니다.</p>
<p>투표 용지는 이 선거의 Python 유권자 명부를 기반으로 배포되었습니다. 이 파일은 개인 이메일 주소를 포함하고 있어 공개되지 않지만, 명부가 생성되면 이름으로 된 "Complete Voter Roll"이 공개될 예정이었습니다.</p>
<h2>선거 구현 (Election Implementation)</h2>
<p>선거는 Helios Voting Service를 사용하여 진행되었습니다.</p>
<p><strong>Configuration (구성):</strong></p>
<ul>
<li><strong>Short name:</strong> <code>2023-python-steering-council</code></li>
<li><strong>Name:</strong> <code>2023 Python Steering Council Election</code></li>
<li><strong>Description:</strong> PEP 13에 명시된 Python 스티어링 카운슬 선거. 2023년 임기 스티어링 카운슬 선거입니다.</li>
<li><strong>type:</strong> <code>Election</code></li>
<li><strong>Use voter aliases:</strong> <code>[X]</code> (유권자 별칭 사용)</li>
<li><strong>Randomize answer order:</strong> <code>[X]</code> (답변 순서 무작위화)</li>
<li><strong>Private:</strong> <code>[X]</code> (비공개)</li>
<li><strong>Help Email Address:</strong> <code>psf-election@python.org</code></li>
<li><strong>Voting starts at:</strong> 2022년 12월 1일 12:00 UTC</li>
<li><strong>Voting ends at:</strong> 2022년 12월 15일 12:00 UTC</li>
</ul>
<p>이는 다음과 같은 방식으로 선거가 진행됨을 의미합니다:</p>
<ul>
<li>투표는 일반에 공개되지 않으며, 유권자 명부에 있는 사람만 참여할 수 있습니다.</li>
<li>투표 시작 시 투표 용지가 이메일로 발송됩니다.</li>
<li>후보자는 편향을 피하기 위해 무작위 순서로 제시됩니다.</li>
<li>유권자의 신원과 투표 용지는 암호화 기술의 발전으로부터 보호됩니다.</li>
</ul>
<p><strong>Questions (질문):</strong>
<strong>질문 1:</strong> Python 스티어링 카운슬 후보자를 선택하십시오.</p>
<ul>
<li><strong>Select between 0 and - (approval) answers.</strong> (0개에서 -개까지 (승인) 답변 선택)</li>
<li><strong>Result Type:</strong> <code>absolute</code></li>
<li><strong>Answer #1 - #N:</strong> <code>Candidates from Candidates_ Section</code> (후보자 섹션의 후보자들)</li>
</ul>
