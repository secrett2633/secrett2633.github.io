<h1>PEP 628 – <code>math.tau</code> 추가 제안</h1>
<h2>개요</h2>
<p>이 문서는 Python 표준 라이브러리에 원 상수 <code>math.tau</code>를 추가할 것을 제안합니다. <code>tau (τ)</code>의 개념은 원의 둘레(circumference)와 반지름(radius)의 비율이 둘레와 지름(diameter)의 비율보다 훨씬 더 근본적이고 흥미롭다는 관찰에 기반합니다. <code>tau</code>는 단순히 <code>2 * pi (2π)</code> 값에 이름을 할당한 것입니다.</p>
<h2>PEP 수락</h2>
<p>이 PEP는 수락되었으며, <code>math.tau</code>는 Python 3.6에 포함되었습니다. 이 아이디어는 이슈 12345를 통해 구현되었습니다.</p>
<h2><code>tau</code> 도입의 배경</h2>
<p><code>pi (π)</code>는 원의 둘레와 지름의 비율로 정의됩니다. 하지만 원은 중심점과 반지름에 의해 정의됩니다. 이는 원의 둘레에서 넓이(area)로 이동하는 통합(integration) 매개변수가 지름이 아닌 반지름이라는 점을 통해 명확히 드러납니다. 만약 지름을 사용한다면 불필요한 곱셈 계수(extraneous multiplier)를 제거하기 위해 4로 나누어야 합니다.</p>
<p>라디안(radians)을 사용하여 작업할 때, 원의 특정 부분을 <code>tau</code>의 관점에서 라디안 값으로 변환하는 것은 매우 간단합니다.</p>
<ul>
<li>4분의 1원은 <code>tau/4</code></li>
<li>반원은 <code>tau/2</code></li>
<li>25분의 7은 <code>7*tau/25</code></li>
</ul>
<p>이와 대조적으로 <code>pi</code>를 사용하는 동등한 표현(<code>pi/2</code>, <code>pi</code>, <code>14*pi/25</code>)에서는 불필요하고 혼란스러운 2의 곱셈이 사라집니다.</p>
<p><code>tau</code>를 사용함으로써 수학의 많은 측면이 <code>pi</code>를 사용할 때보다 훨씬 더 쉽고 합리적으로 변한다는 점을 지적하는 수많은 사례들이 있습니다.</p>
