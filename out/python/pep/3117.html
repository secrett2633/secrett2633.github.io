PEP 3117 вҖ“ Postfix type declarations (нӣ„мң„ нғҖмһ… м„ м–ё)

к°ңмҡ” (Abstract)

мқҙ PEPлҠ” Pythonм—җ нӣ„мң„(postfix) нғҖмһ… м„ м–ё л¬ёлІ•мқ„ м¶”к°Җн•  кІғмқ„ м ңм•Ҳн•©лӢҲлӢӨ. лҳҗн•ң, нғҖмһ…кіј м„ м–ёмһҗ(declarator) мӮ¬мқҙмқҳ мғҲлЎңмҡҙ л§Өн•‘мқ„ мғқм„ұн•ҳлҠ” лҚ° мӮ¬мҡ©лҗҳлҠ” мғҲлЎңмҡҙ typedef л¬ё(statement)мқ„ лӘ…мӢңн•©лӢҲлӢӨ.

мқҙ м ңм•Ҳмқҙ мҲҳмҡ©лҗҳл©ҙ Python мӮ¬мҡ©мһҗ кІҪн—ҳмқ„ нҒ¬кІҢ н–ҘмғҒмӢңнӮӨкі , лӢӨлҘё н”„лЎңк·ёлһҳл°Қ м–ём–ҙ мӮ¬мҡ©мһҗл“Өмқҙ PythonмңјлЎң м „нҷҳн•ҳлҠ” кІғмқ„ л§қм„ӨмқҙкІҢ н•ҳлҠ” "лӢЁм җ" мӨ‘ н•ҳлӮҳлҘј м ңкұ°н•  мҲҳ мһҲмқ„ кІғмқҙлқјкі  н•©лӢҲлӢӨ.

лҸ„мһ… л°°кІҪ (Rationale)

PythonмқҖ мҳӨлһ«лҸҷм•Ҳ лӘ…мӢңм Ғмқё нғҖмһ… м„ м–ё(explicit type declarations)мқҳ л¶Җмһ¬лЎң м–ҙл ӨмӣҖмқ„ кІӘм–ҙмҷ”мҠөлӢҲлӢӨ. мқҙлҠ” Pythonмқҳ Zen мӨ‘ м–ём–ҙк°Җ лІ—м–ҙлӮҳлҠ” лӘҮ м•Ҳ лҗҳлҠ” л¶Җл¶„ мӨ‘ н•ҳлӮҳлЎң, Python мқҙлӢЁмһҗл“Ө(heretics)кіј PSU л©ӨлІ„л“Ө мӮ¬мқҙм—җм„ң л§ҺмқҖ л…јмҹҒмқ„ л¶Ҳлҹ¬мқјмңјмј°кі , лҢҖк·ңлӘЁ м—”н„°н”„лқјмқҙмҰҲ м„ұкіөмқ„ м–ҙл өкІҢ л§Ңл“Өм—ҲмҠөлӢҲлӢӨ.

н•ҳм§Җл§Ң мқҙлҹ¬н•ң л¬ём ңлҘј н•ҙкІ°н•ҳл Өл©ҙ "нҢҢмқҙмҚ¬мҠӨлҹ¬мҡҙ(Pythonic)" л¬ёлІ•мқ„ м°ҫм•„м•ј н•©лӢҲлӢӨ. нғҖмһ… м„ м–ёмқ„ к°Җм§„ кұ°мқҳ лӘЁл“  м–ём–ҙм—җм„ң нғҖмһ… м„ м–ёмқҖ мқҙлҹ¬н•ң нҠ№м„ұмқ„ кІ°м—¬н•ҳкі  мһҲмҠөлӢҲлӢӨ. мҰү, мһҘнҷ©н•ҳкі (verbose), лӢЁмқј нғҖмһ…мқ„ мң„н•ҙ м—¬лҹ¬ лӢЁм–ҙлҘј н•„мҡ”лЎң н•ҳкұ°лӮҳ, мқҙн•ҙн•ҳкё° м–ҙл өмҠөлӢҲлӢӨ(мҳҲ: нҠ№м • м–ём–ҙлҠ” нғҖмһ… м„ м–ём—җ dimкіј к°ҷмқҙ мҷ„м „нһҲ кҙҖл Ё м—ҶлҠ” нҳ•мҡ©мӮ¬лҘј мӮ¬мҡ©н•©лӢҲлӢӨ).

л”°лқјм„ң мқҙ PEPлҠ” нғҖмһ… м„ м–ёмңјлЎңмқҳ м „нҷҳкіј н•Ёк»ҳ, Pythonмқҙ лҜёлһҳм—җ лҢҖл№„н•  лҝҗл§Ң м•„лӢҲлқј лҜёлһҳлҘј нҸ¬мҡ©н•ңлӢӨлҠ” кІғмқ„ лӢӨмӢң н•ңлІҲ мҰқлӘ…н•  лҳҗ лӢӨлҘё кіјк°җн•ң мӣҖм§Ғмһ„мқ„ кІ°н•©н•©лӢҲлӢӨ. к·ёкІғмқҖ л°”лЎң мң лӢҲмҪ”л“ң л¬ёмһҗ(Unicode characters)лҘј мҶҢмҠӨ мҪ”л“ңмқҳ н•„мҲҳм Ғмқё кө¬м„ұ мҡ”мҶҢлЎң лҸ„мһ…н•ҳлҠ” кІғмһ…лӢҲлӢӨ.

мң лӢҲмҪ”л“ңлҠ” нӣЁм”¬ м ҒмқҖ л¬ёмһҗлЎң нӣЁм”¬ лҚ” л§ҺмқҖ кІғмқ„ н‘ңнҳ„н•  мҲҳ мһҲкІҢ н•ҙмЈјл©°, мқҙлҠ” Pythonмқҳ Zenмқё "к°ҖлҸ…м„ұмқҙ мӨ‘мҡ”н•©лӢҲлӢӨ(Readability counts)"м—җ л¶Җн•©н•©лӢҲлӢӨ. лҳҗн•ң, лі„лҸ„мқҳ нғҖмһ… м„ м–ёл¬ёмқҙ н•„мҡ” м—ҶкІҢ н•ҳл©°, л§Ҳм§Җл§үмңјлЎң Perl 6к°Җ мқҙлҜё м—°мӮ°мһҗм—җ мң лӢҲмҪ”л“ңлҘј мӮ¬мҡ©н•ҳкі  мһҲлӢӨлҠ” м җмқ„ к°җм•Ҳн•  л•Ң Pythonмқҙ Perl 6м—җ н•„м Ғн•ҳкІҢ л§Ңл“ӯлӢҲлӢӨ. [2, cite: 1]

лӘ…м„ё (Specification)

нғҖмһ… м„ м–ё лӘЁл“ңк°Җ нҷңм„ұнҷ”лҗҳл©ҙ, л¬ёлІ•мқҙ ліҖкІҪлҗҳм–ҙ к°Ғ NAMEмқҖ мқҙлҰ„(name)кіј нғҖмһ… м„ м–ёмһҗ(type declarator)мқҳ л‘җ л¶Җл¶„мңјлЎң кө¬м„ұлҗҳм–ҙм•ј н•ҳл©°, мқҙ нғҖмһ… м„ м–ёмһҗлҠ” м •нҷ•нһҲ н•ҳлӮҳмқҳ мң лӢҲмҪ”л“ң л¬ёмһҗмһ…лӢҲлӢӨ.

м„ м–ёмһҗлҠ” мқҙлҰ„мқҳ нғҖмһ…мқ„ кі мң н•ҳкІҢ м§Җм •н•ҳл©°, л§Ңм•Ҫ н‘ңнҳ„мӢқмқҳ мҷјмӘҪм—җ лӮҳнғҖлӮҳл©ҙ мқҙ нғҖмһ…мқҙ к°•м ңлҗ©лӢҲлӢӨ. л°ҳнҷҳлҗң нғҖмһ…мқҙ м„ м–ёлҗң нғҖмһ…кіј мқјм№ҳн•ҳм§Җ м•Ҡмңјл©ҙ InquisitionError мҳҲмҷёк°Җ л°ңмғқн•©лӢҲлӢӨ. [3, cite: 1]

лҳҗн•ң, н•ЁмҲҳ нҳём¶ң кІ°кіј нғҖмһ…лҸ„ м§Җм •н•ҙм•ј н•©лӢҲлӢӨ. нҳём¶ң кІ°кіјк°Җ м„ м–ёлҗң нғҖмһ…кіј лӢӨлҘҙл©ҙ InquisitionErrorк°Җ л°ңмғқн•©лӢҲлӢӨ. мЈјмқҳ: кІ°кіјм—җ лҢҖн•ң м„ м–ёмһҗлҘј н•ЁмҲҳ к°қмІҙм—җ лҢҖн•ң м„ м–ёмһҗмҷҖ нҳјлҸҷн•ҙм„ңлҠ” м•Ҳ лҗ©лӢҲлӢӨ(м•„лһҳ мҳҲмӢң м°ёмЎ°).

мқҪкё°л§Ң н•ҳкі  н• лӢ№лҗҳм§Җ м•ҠлҠ” мқҙлҰ„ л’Өмқҳ нғҖмһ… м„ м–ёмһҗлҠ” м—„кІ©н•ҳкІҢ н•„мҡ”н•ҳм§ҖлҠ” м•Ҡм§Җл§Ң, м–ҙмЁҢл“  к°•м ңлҗ©лӢҲлӢӨ(Python Zen: "лӘ…мӢңм Ғмқё кІғмқҙ м•”мӢңм Ғмқё кІғліҙлӢӨ мўӢмҠөлӢҲлӢӨ(Explicit is better than implicit)").

нғҖмһ…кіј м„ м–ёмһҗ к°„мқҳ л§Өн•‘мқҖ кі м •лҗҳм–ҙ мһҲм§Җ м•ҠмҠөлӢҲлӢӨ. н”„лЎңк·ёлһҳлЁёк°Җ мҷ„м „нһҲ мӮ¬мҡ©мһҗ м •мқҳн•  мҲҳ мһҲм§Җл§Ң, нҺёмқҳлҘј мң„н•ҙ мқјл¶Җ лӮҙмһҘ нғҖмһ…м—җ лҢҖн•ң лҜёлҰ¬ м •мқҳлҗң л§Өн•‘мқҙ мһҲмҠөлӢҲлӢӨ.

мқҙлҹ¬н•ң л¬ёмһҗл“ӨмқҖ лӘЁл“  н”„лЎңк·ёлһҳлЁём—җкІҢ лӘ…нҷ•н•ҳкі  кё°м–өн•ҳкё° мү¬мҡ°л©° мһ…л Ҙн•ҳкё° мү¬мҡё кІғмқҙлқјкі  м–ёкёүлҗ©лӢҲлӢӨ.

мң лӢҲмҪ”л“ң лҢҖмІҙ лӢЁмң„ (Unicode replacement units)

нҳ„лҢҖм Ғмқҙкі  кёҖлЎңлІҢнҷ”лҗң м„ёмғҒм—җм„ңлҸ„ м—¬м „нһҲ мҶҢмҠӨ мҪ”л“ңм—җ мң лӢҲмҪ”л“ңлҘј мӮ¬мҡ©н•ҳм§Җ м•Ҡкұ°лӮҳ мӮ¬мҡ©н•ҳкі  мӢ¶м§Җ м•ҠмқҖ кө¬мӢқ л°ҳн•ӯм•„л“Өмқҙ мһҲмңјл©°, PythonмқҖ кҙҖмҡ©м Ғмқё м–ём–ҙмқҙлҜҖлЎң, мқҙлҹ¬н•ң мӮ¬лһҢл“Өмқ„ мң„н•ң лҢҖмІҙ(fallback)к°Җ м ңкіөлҗ©лӢҲлӢӨ.

лӢЁмқј мң лӢҲмҪ”л“ң л¬ёмһҗ лҢҖмӢ , name${UNICODE NAME OF THE DECLARATOR}$лҘј мһ…л Ҙн•  мҲҳ мһҲмҠөлӢҲлӢӨ. мҳҲлҘј л“Өм–ҙ, лӢӨмқҢ л‘җ н•ЁмҲҳ м •мқҳлҠ” лҸҷмқјн•©лӢҲлӢӨ.

def fooЖӣ(xв„Ӯ): return None

к·ёлҰ¬кі 

def foo${LATIN SMALL LETTER LAMBDA WITH STROKE}$(x${DOUBLE-STRUCK CAPITAL C}$): return None${ZERO WIDTH NO-BREAK SPACE}$

мқҙлҠ” м—¬м „нһҲ мқҪкё° мү¬мҡ°л©°, нғҖмһ… мЈјм„қ(type-annotated)мқҙ м Ғмҡ©лҗң Pythonмқҳ лӘЁл“  кё°лҠҘмқ„ ASCII мӢ лҙүмһҗл“Өм—җкІҢлҸ„ м ңкіөн•©лӢҲлӢӨ.

typedef л¬ё (The typedef statement)

нғҖмһ…кіј м„ м–ёмһҗ к°„мқҳ л§Өн•‘мқҖ мқҙ мғҲлЎңмҡҙ л¬ёмқ„ нҶөн•ҙ нҷ•мһҘлҗ  мҲҳ мһҲмҠөлӢҲлӢӨ.

кө¬л¬ёмқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

typedef_stmt ::= "typedef" expr DECLARATOR

м—¬кё°м„ң exprмқҖ нғҖмһ… к°қмІҙлЎң н•ҙм„қлҗ©лӢҲлӢӨ. нҺёмқҳлҘј мң„н•ҙ typedef л¬ёмқҖ мғҲ нҒҙлһҳмҠӨмқҳ class л¬ёкіјлҸ„ нҳјн•©н•ҳм—¬ мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

typedef class Fooвҳә(objectпҝҪ): pass

мҳҲмӢң (Example)

лӢӨмқҢмқҖ н‘ңмӨҖ os.path.normpath н•ЁмҲҳлҘј нғҖмһ… м„ м–ё л¬ёлІ•мңјлЎң ліҖнҷҳн•ң кІғмһ…лӢҲлӢӨ.

def normpathЖӣ(pathвңҺ)вңҺ:
    """Normalize path, eliminating double slashes, etc."""
    if pathвңҺ == '':
        return '.'
    initial_slashesвң“ = pathвңҺ.startswithЖӣ('/')вң“ # POSIX allows one or two initial slashes, but treats three or more
                                                # as single slash.
    if (initial_slashesвң“ and pathвңҺ.startswithЖӣ('//')вң“ and not pathвңҺ.startswithЖӣ('///')вң“)вң“:
        initial_slashesв„• = 2
    compsвҷЁ = pathвңҺ.splitЖӣ('/')вҷЁ
    new_compsвҷЁ = []вҷЁ
    for compвңҺ in compsвҷЁ:
        if compвңҺ in ('', '.')в’Ҝ:
            continue
        if (compвңҺ != '..' or (not initial_slashesв„• and not new_compsвҷЁ)вң“ or
                (new_compsвҷЁ and new_compsвҷЁ[-1]вңҺ == '..')вң“)вң“:
            new_compsвҷЁ.appendЖӣ(compвңҺ)
        elif new_compsвҷЁ:
            new_compsвҷЁ.popЖӣ()вңҺ
    compsвҷЁ = new_compsвҷЁ
    pathвңҺ = '/'.join(compsвҷЁ)вңҺ
    if initial_slashesв„•:
        pathвңҺ = '/'*initial_slashesв„• + pathвңҺ
    return pathвңҺ or '.'

ліҙмӢңлӢӨмӢңн”ј, нғҖмһ… м„ м–ёмқҖ н‘ңнҳ„л Ҙмқ„ лҚ”н•ҳл©ҙм„ң лҸҷмӢңм—җ мҪ”л“ңлҘј нӣЁм”¬ лҚ” м „л¬ём ҒмңјлЎң ліҙмқҙкІҢ н•©лӢҲлӢӨ.

нҳёнҷҳм„ұ л¬ём ң (Compatibility issues)

нғҖмһ… м„ м–ё лӘЁл“ңлҘј нҷңм„ұнҷ”н•ҳл Өл©ҙ лӢӨмқҢмқ„ мһ‘м„ұн•ҙм•ј н•©лӢҲлӢӨ.

from __future__ import type_declarations

мқҙлҠ” мҶҢмҠӨмқҳ мң лӢҲмҪ”л“ң нҢҢмӢұмқ„ нҷңм„ұнҷ”н•ҳкі  [4, cite: 1], typedefлҘј нӮӨмӣҢл“ңлЎң л§Ңл“Өл©° лӘЁл“  н• лӢ№ л°Ҹ н•ЁмҲҳ нҳём¶ңм—җ лҢҖн•ҙ мҳ¬л°”лҘё нғҖмһ…мқ„ к°•м ңн•©лӢҲлӢӨ.

мөңмў… кІ°м • (Rejection)

мӢ мӨ‘н•ң кі л Ө, л§ҺмқҖ мһҗкё° м„ұм°°, кі лҮҢ лҒқм—җ мқҙ PEPлҠ” мөңмў…м ҒмңјлЎң кұ°л¶Җ(Rejected) лҗҳм—ҲмҠөлӢҲлӢӨ.

## к°ңмҡ” (Abstract)

мқҙ PEPлҠ” Pythonм—җ нӣ„мң„(postfix) нғҖмһ… м„ м–ё л¬ёлІ•мқ„ м¶”к°Җн•  кІғмқ„ м ңм•Ҳн•©лӢҲлӢӨ. лҳҗн•ң, нғҖмһ…кіј м„ м–ёмһҗ(declarator) мӮ¬мқҙмқҳ мғҲлЎңмҡҙ л§Өн•‘мқ„ мғқм„ұн•ҳлҠ” лҚ° мӮ¬мҡ©лҗҳлҠ” мғҲлЎңмҡҙ `typedef` л¬ё(statement)мқ„ лӘ…мӢңн•©лӢҲлӢӨ.

мқҙ м ңм•Ҳмқҙ мҲҳмҡ©лҗҳл©ҙ Python мӮ¬мҡ©мһҗ кІҪн—ҳмқ„ нҒ¬кІҢ н–ҘмғҒмӢңнӮӨкі , лӢӨлҘё н”„лЎңк·ёлһҳл°Қ м–ём–ҙ мӮ¬мҡ©мһҗл“Өмқҙ PythonмңјлЎң м „нҷҳн•ҳлҠ” кІғмқ„ л§қм„ӨмқҙкІҢ н•ҳлҠ” "лӢЁм җ" мӨ‘ н•ҳлӮҳлҘј м ңкұ°н•  мҲҳ мһҲмқ„ кІғмқҙлқјкі  н•©лӢҲлӢӨ.

## лҸ„мһ… л°°кІҪ (Rationale)

PythonмқҖ мҳӨлһ«лҸҷм•Ҳ лӘ…мӢңм Ғмқё нғҖмһ… м„ м–ё(explicit type declarations)мқҳ л¶Җмһ¬лЎң м–ҙл ӨмӣҖмқ„ кІӘм–ҙмҷ”мҠөлӢҲлӢӨ. мқҙлҠ” Pythonмқҳ Zen мӨ‘ м–ём–ҙк°Җ лІ—м–ҙлӮҳлҠ” лӘҮ м•Ҳ лҗҳлҠ” л¶Җл¶„ мӨ‘ н•ҳлӮҳлЎң, Python мқҙлӢЁмһҗл“Ө(heretics)кіј PSU л©ӨлІ„л“Ө мӮ¬мқҙм—җм„ң л§ҺмқҖ л…јмҹҒмқ„ л¶Ҳлҹ¬мқјмңјмј°кі , лҢҖк·ңлӘЁ м—”н„°н”„лқјмқҙмҰҲ м„ұкіөмқ„ м–ҙл өкІҢ л§Ңл“Өм—ҲмҠөлӢҲлӢӨ.

н•ҳм§Җл§Ң мқҙлҹ¬н•ң л¬ём ңлҘј н•ҙкІ°н•ҳл Өл©ҙ "нҢҢмқҙмҚ¬мҠӨлҹ¬мҡҙ(Pythonic)" л¬ёлІ•мқ„ м°ҫм•„м•ј н•©лӢҲлӢӨ. нғҖмһ… м„ м–ёмқ„ к°Җм§„ кұ°мқҳ лӘЁл“  м–ём–ҙм—җм„ң нғҖмһ… м„ м–ёмқҖ мқҙлҹ¬н•ң нҠ№м„ұмқ„ кІ°м—¬н•ҳкі  мһҲмҠөлӢҲлӢӨ. мҰү, мһҘнҷ©н•ҳкі (verbose), лӢЁмқј нғҖмһ…мқ„ мң„н•ҙ м—¬лҹ¬ лӢЁм–ҙлҘј н•„мҡ”лЎң н•ҳкұ°лӮҳ, мқҙн•ҙн•ҳкё° м–ҙл өмҠөлӢҲлӢӨ(мҳҲ: нҠ№м • м–ём–ҙлҠ” нғҖмһ… м„ м–ём—җ `dim`кіј к°ҷмқҙ мҷ„м „нһҲ кҙҖл Ё м—ҶлҠ” нҳ•мҡ©мӮ¬лҘј мӮ¬мҡ©н•©лӢҲлӢӨ).

л”°лқјм„ң мқҙ PEPлҠ” нғҖмһ… м„ м–ёмңјлЎңмқҳ м „нҷҳкіј н•Ёк»ҳ, Pythonмқҙ лҜёлһҳм—җ лҢҖл№„н•  лҝҗл§Ң м•„лӢҲлқј лҜёлһҳлҘј нҸ¬мҡ©н•ңлӢӨлҠ” кІғмқ„ лӢӨмӢң н•ңлІҲ мҰқлӘ…н•  лҳҗ лӢӨлҘё кіјк°җн•ң мӣҖм§Ғмһ„мқ„ кІ°н•©н•©лӢҲлӢӨ. к·ёкІғмқҖ л°”лЎң мң лӢҲмҪ”л“ң л¬ёмһҗ(Unicode characters)лҘј мҶҢмҠӨ мҪ”л“ңмқҳ н•„мҲҳм Ғмқё кө¬м„ұ мҡ”мҶҢлЎң лҸ„мһ…н•ҳлҠ” кІғмһ…лӢҲлӢӨ.

мң лӢҲмҪ”л“ңлҠ” нӣЁм”¬ м ҒмқҖ л¬ёмһҗлЎң нӣЁм”¬ лҚ” л§ҺмқҖ кІғмқ„ н‘ңнҳ„н•  мҲҳ мһҲкІҢ н•ҙмЈјл©°, мқҙлҠ” Pythonмқҳ Zenмқё "к°ҖлҸ…м„ұмқҙ мӨ‘мҡ”н•©лӢҲлӢӨ(Readability counts)"м—җ л¶Җн•©н•©лӢҲлӢӨ. лҳҗн•ң, лі„лҸ„мқҳ нғҖмһ… м„ м–ёл¬ёмқҙ н•„мҡ” м—ҶкІҢ н•ҳл©°, л§Ҳм§Җл§үмңјлЎң Perl 6к°Җ мқҙлҜё м—°мӮ°мһҗм—җ мң лӢҲмҪ”л“ңлҘј мӮ¬мҡ©н•ҳкі  мһҲлӢӨлҠ” м җмқ„ к°җм•Ҳн•  л•Ң Pythonмқҙ Perl 6м—җ н•„м Ғн•ҳкІҢ л§Ңл“ӯлӢҲлӢӨ. [2, cite: 1]

## лӘ…м„ё (Specification)

нғҖмһ… м„ м–ё лӘЁл“ңк°Җ нҷңм„ұнҷ”лҗҳл©ҙ, л¬ёлІ•мқҙ ліҖкІҪлҗҳм–ҙ к°Ғ `NAME`мқҖ мқҙлҰ„(name)кіј нғҖмһ… м„ м–ёмһҗ(type declarator)мқҳ л‘җ л¶Җл¶„мңјлЎң кө¬м„ұлҗҳм–ҙм•ј н•ҳл©°, мқҙ нғҖмһ… м„ м–ёмһҗлҠ” м •нҷ•нһҲ н•ҳлӮҳмқҳ мң лӢҲмҪ”л“ң л¬ёмһҗмһ…лӢҲлӢӨ.

м„ м–ёмһҗлҠ” мқҙлҰ„мқҳ нғҖмһ…мқ„ кі мң н•ҳкІҢ м§Җм •н•ҳл©°, л§Ңм•Ҫ н‘ңнҳ„мӢқмқҳ мҷјмӘҪм—җ лӮҳнғҖлӮҳл©ҙ мқҙ нғҖмһ…мқҙ к°•м ңлҗ©лӢҲлӢӨ. л°ҳнҷҳлҗң нғҖмһ…мқҙ м„ м–ёлҗң нғҖмһ…кіј мқјм№ҳн•ҳм§Җ м•Ҡмңјл©ҙ `InquisitionError` мҳҲмҷёк°Җ л°ңмғқн•©лӢҲлӢӨ. [3, cite: 1]

лҳҗн•ң, н•ЁмҲҳ нҳём¶ң кІ°кіј нғҖмһ…лҸ„ м§Җм •н•ҙм•ј н•©лӢҲлӢӨ. нҳём¶ң кІ°кіјк°Җ м„ м–ёлҗң нғҖмһ…кіј лӢӨлҘҙл©ҙ `InquisitionError`к°Җ л°ңмғқн•©лӢҲлӢӨ. мЈјмқҳ: кІ°кіјм—җ лҢҖн•ң м„ м–ёмһҗлҘј н•ЁмҲҳ к°қмІҙм—җ лҢҖн•ң м„ м–ёмһҗмҷҖ нҳјлҸҷн•ҙм„ңлҠ” м•Ҳ лҗ©лӢҲлӢӨ(м•„лһҳ мҳҲмӢң м°ёмЎ°).

мқҪкё°л§Ң н•ҳкі  н• лӢ№лҗҳм§Җ м•ҠлҠ” мқҙлҰ„ л’Өмқҳ нғҖмһ… м„ м–ёмһҗлҠ” м—„кІ©н•ҳкІҢ н•„мҡ”н•ҳм§ҖлҠ” м•Ҡм§Җл§Ң, м–ҙмЁҢл“  к°•м ңлҗ©лӢҲлӢӨ(Python Zen: "лӘ…мӢңм Ғмқё кІғмқҙ м•”мӢңм Ғмқё кІғліҙлӢӨ мўӢмҠөлӢҲлӢӨ(Explicit is better than implicit)").

### мң лӢҲмҪ”л“ң лҢҖмІҙ лӢЁмң„ (Unicode replacement units)

нҳ„лҢҖм Ғмқҙкі  кёҖлЎңлІҢнҷ”лҗң м„ёмғҒм—җм„ңлҸ„ м—¬м „нһҲ мҶҢмҠӨ мҪ”л“ңм—җ мң лӢҲмҪ”л“ңлҘј мӮ¬мҡ©н•ҳм§Җ м•Ҡкұ°лӮҳ мӮ¬мҡ©н•ҳкі  мӢ¶м§Җ м•ҠмқҖ кө¬мӢқ л°ҳн•ӯм•„л“Өмқҙ мһҲмңјл©°, PythonмқҖ кҙҖмҡ©м Ғмқё м–ём–ҙмқҙлҜҖлЎң, мқҙлҹ¬н•ң мӮ¬лһҢл“Өмқ„ мң„н•ң лҢҖмІҙ(fallback)к°Җ м ңкіөлҗ©лӢҲлӢӨ.
