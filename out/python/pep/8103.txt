<h1 id="pep-8103-2022년도-steering-council-선거"><a href="#pep-8103-2022년도-steering-council-선거">PEP 8103 – 2022년도 Steering Council 선거</a></h1>
<ul>
<li><strong>작성자:</strong> Ewa Jodlowska, Ee Durbin, Joe Carey</li>
<li><strong>후원자:</strong> Barry Warsaw</li>
<li><strong>상태:</strong> Final</li>
<li><strong>유형:</strong> Informational (정보성)</li>
<li><strong>주제:</strong> Governance (거버넌스)</li>
<li><strong>생성일:</strong> 2021년 10월 4일</li>
</ul>
<h2 id="개요-abstract"><a href="#개요-abstract">개요 (Abstract)</a></h2>
<p>이 문서는 PEP 13에 명시된 대로 2021년 12월에 진행된 Python Steering Council (운영 위원회) 선거의 일정 및 기타 세부 사항을 설명합니다. 이는 2022년 임기(즉, Python 3.11 개발 기간)를 위한 Steering Council 선거입니다.</p>
<h2 id="선거-관리-election-administration"><a href="#선거-관리-election-administration">선거 관리 (Election Administration)</a></h2>
<p>Steering Council은 Python Software Foundation의 인프라 디렉터인 Ee Durbin과 회계 관리자인 Joe Carey를 선거 공동 관리자로 임명했습니다.</p>
<h2 id="일정-schedule"><a href="#일정-schedule">일정 (Schedule)</a></h2>
<p>지명 기간(nomination period)은 2주, 투표 기간(voting period)은 2주로 진행되었습니다.</p>
<ul>
<li><strong>지명 기간:</strong> 2021년 11월 1일부터 2021년 11월 16일 12:00 UTC (Anywhere on Earth 기준 2021년 11월 15일 말)까지였습니다. 이 내용은 <code>python-committers</code> 메일링 리스트에 공지되었고, <code>discuss.python.org</code>에서 진행되었습니다.</li>
<li><strong>투표 기간:</strong> 2021년 12월 1일 12:00 UTC부터 2021년 12월 16일 12:00 UTC (Anywhere on Earth 기준 2021년 12월 15일 말)까지였습니다.</li>
</ul>
<h2 id="후보자-candidates"><a href="#후보자-candidates">후보자 (Candidates)</a></h2>
<p>후보자는 코어 팀 멤버에 의해 지명되어야 합니다. 후보자가 코어 팀 멤버인 경우 스스로를 지명할 수 있습니다.</p>
<p>지명된 후보자(알파벳 순):</p>
<ul>
<li>Brett Cannon</li>
<li>Mariatta</li>
<li>David Mertz</li>
<li>Dong-hee Na (나동희)</li>
<li>Pablo Galindo Salgado</li>
<li>Gregory P. Smith</li>
<li>Victor Stinner</li>
<li>Petr Viktorin</li>
<li>Barry Warsaw</li>
<li>Thomas Wouters</li>
</ul>
<p>철회된 지명은 없었습니다.</p>
<h2 id="유권자-명부-voter-roll"><a href="#유권자-명부-voter-roll">유권자 명부 (Voter Roll)</a></h2>
<p>모든 활동 중인 Python 코어 팀 멤버는 투표 자격이 있습니다. 활동 상태는 PEP 13에 설명된 대로 결정되며, <code>python/voters</code> 소프트웨어를 통해 구현됩니다.</p>
<p>투표 용지는 이 선거를 위한 Python 유권자 명부(<code>The Python Voter Roll</code>)를 기반으로 배포되었습니다.
이 파일은 개인 이메일 주소를 포함하므로 공개되지 않지만, 명부가 생성되면 이름으로 된 전체 유권자 명부는 공개될 예정이었습니다.</p>
<h2 id="선거-구현-election-implementation"><a href="#선거-구현-election-implementation">선거 구현 (Election Implementation)</a></h2>
<p>선거는 Helios Voting Service를 사용하여 진행되었습니다.</p>
<h3 id="설정-configuration"><a href="#설정-configuration">설정 (Configuration)</a></h3>
<ul>
<li><strong>Short name:</strong> <code>2022-python-steering-council</code></li>
<li><strong>Name:</strong> <code>2022 Python Steering Council Election</code></li>
<li><strong>Description:</strong> PEP 13에 명시된 Python Steering Council 선거입니다. 2022년 임기를 위한 Steering Council 선거입니다.</li>
<li><strong>Type:</strong> Election</li>
<li><strong>Use voter aliases:</strong> [X] (유권자 가명 사용)</li>
<li><strong>Randomize answer order:</strong> [X] (답변 순서 무작위화)</li>
<li><strong>Private:</strong> [X] (비공개)</li>
<li><strong>Help Email Address:</strong> <code>psf-election@python.org</code></li>
<li><strong>Voting starts at:</strong> 2021년 12월 1일 00:00 UTC</li>
<li><strong>Voting ends at:</strong> 2021년 12월 16일 12:00 UTC</li>
</ul>
<p>이러한 설정으로 다음과 같은 선거가 진행되었습니다.</p>
<ul>
<li>투표는 일반에 공개되지 않으며, 유권자 명부에 있는 사람만 참여할 수 있습니다.</li>
<li>투표 시작 시 투표 용지가 이메일로 발송됩니다.</li>
<li>후보자들은 편향을 피하기 위해 무작위 순서로 제시됩니다.</li>
<li>유권자의 신원과 투표 용지는 암호학적 공격으로부터 보호됩니다.</li>
</ul>
<h3 id="질문-questions"><a href="#질문-questions">질문 (Questions)</a></h3>
<p><strong>질문 1</strong></p>
<ul>
<li><strong>Select between 0 and - (approval) answers.</strong> (0개에서 승인 가능한 범위 내에서 답변을 선택하세요.)</li>
<li><strong>Result Type:</strong> absolute (절대값)</li>
<li><strong>Question:</strong> Select candidates for the Python Steering Council (Python Steering Council 후보자를 선택하세요)</li>
<li><strong>Answer #1 - #N:</strong> Candidates from Candidates_ Section (후보자 섹션의 후보자들)</li>
</ul>
