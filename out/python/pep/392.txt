<h2 id="pep-392-python-32-릴리즈-일정-번역-및-정리"><a href="#pep-392-python-32-릴리즈-일정-번역-및-정리">PEP 392 – Python 3.2 릴리즈 일정 번역 및 정리</a></h2>
<p>본 문서는 Python 3.2 시리즈의 개발 및 릴리즈 일정을 설명합니다. 이 일정은 주로 PEP 규모의 항목들을 다룹니다.</p>
<h3 id="개요-abstract"><a href="#개요-abstract">개요 (Abstract)</a></h3>
<p>이 문서는 Python 3.2 시리즈의 개발 및 릴리즈 일정을 설명합니다. 이 일정은 주로 PEP(Python Enhancement Proposal) 규모의 항목들에 중점을 둡니다.</p>
<h3 id="릴리즈-관리자-및-팀-release-manager-and-crew"><a href="#릴리즈-관리자-및-팀-release-manager-and-crew">릴리즈 관리자 및 팀 (Release Manager and Crew)</a></h3>
<ul>
<li><strong>3.2 릴리즈 관리자 (Release Manager):</strong> Georg Brandl</li>
<li><strong>Windows 설치 프로그램:</strong> Martin v. Loewis</li>
<li><strong>Mac 설치 프로그램:</strong> Ronald Oussoren</li>
<li><strong>문서화 (Documentation):</strong> Georg Brandl</li>
</ul>
<h3 id="32-수명-주기-32-lifespan"><a href="#32-수명-주기-32-lifespan">3.2 수명 주기 (3.2 Lifespan)</a></h3>
<p>Python 3.2는 약 18개월 동안 4~6개월마다 버그 수정 업데이트를 받았습니다. 3.3.0 최종 버전(PEP 398 참조)이 릴리즈된 후, 최종 3.2 버그 수정 업데이트가 릴리즈되었습니다. 그 이후에는 3.2 최종 버전 릴리즈 후 5년까지(원래 2016년 2월로 계획됨) 보안 업데이트(소스 코드만 해당)가 릴리즈되었습니다.</p>
<p>2016년 2월 20일을 기준으로 Python 3.2.x는 EOL(End-Of-Life) 상태에 도달했습니다. 최종 소스 릴리즈는 2014년 10월의 3.2.6 버전이었습니다.</p>
