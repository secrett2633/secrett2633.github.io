кұ°л¶Җ кіөм§Җ (Rejection Notice)

мқҙ PEPлҠ” Raymond Hettingerмқҳ мЎ°м–ём—җ л”°лқј кұ°л¶Җлҗҳм—ҲмҠөлӢҲлӢӨ.

Raymond HettingerлҠ” BListмқҳ мҶҢмҠӨ мҪ”л“ңлҘј кІҖнҶ н•ң нӣ„, кё°мЎҙ list()лҘј лҢҖмІҙн•  к°ҖлҠҘм„ұмқҙ кұ°мқҳ м—ҶлӢӨкі  нҢҗлӢЁн–ҲмҠөлӢҲлӢӨ. к·ё мқҙмң лҠ” listк°Җ лӢЁмҲңн•ң C API, мһ‘мқҖ лҰ¬мҠӨнҠём—җ лҢҖн•ң лӮ®мқҖ кіөк°„ мҳӨлІ„н—Өл“ң, мқјл°ҳм Ғмқё мӮ¬мҡ© мӮ¬лЎҖм—җм„ңмқҳ мҡ°мҲҳн•ң м„ұлҠҘ, к·ёлҰ¬кі  мүҪкІҢ мқҙн•ҙн•  мҲҳ мһҲлҠ” м„ұлҠҘ нҠ№м„ұм—җм„ң мҳӨлҠ” мқҙм җмқҙ л§Өмҡ° нҒ¬кё° л•Ңл¬ёмһ…лӢҲлӢӨ. BList кө¬нҳ„мқҖ мқҙлҹ¬н•ң мһҘм җл“Өмқҙ л¶ҖмЎұн•ҳл©°, мқјл°ҳм Ғмқё кІҪмҡ°мқҳ м„ұлҠҘмқ„ м•Ҫк°„ нқ¬мғқн•ҳкі  нқ”м№ҳ м•ҠмқҖ кІҪмҡ°мқҳ м„ұлҠҘмқ„ нҒ¬кІҢ к°ңм„ н•ҳлҠ” нҠёл Ҳмқҙл“ңмҳӨн”„лҘј к°Җм§Җкі  мһҲмҠөлӢҲлӢӨ. л”°лқјм„ң Py3.0 PEPлЎңм„ң кұ°л¶Җлҗ  мҲҳ мһҲлӢӨкі  ліҙм•ҳмҠөлӢҲлӢӨ.

лӢӨл§Ң, BListк°Җ м„ңл“ңнҢҢнӢ° лӘЁл“ҲлЎңм„ң м„ұкіөн•ңлӢӨл©ҙ collections лӘЁл“Ҳм—җ нҸ¬н•Ёлҗ  к°ҖлҠҘм„ұмқҖ м—¬м „нһҲ лӮЁм•„мһҲлӢӨкі  м–ёкёүн–ҲмҠөлӢҲлӢӨ. мқҙлҘј мң„н•ң н•өмӢ¬ кё°мӨҖмқҖ BListк°Җ нҠ№м • мӢӨм ң мӮ¬мҡ© мӮ¬лЎҖм—җм„ң кё°мЎҙ listліҙлӢӨ мҡ°мӣ”н•ң м„ нғқмқҙ лҗҳлҠ”м§Җ м—¬л¶Җмһ…лӢҲлӢӨ. Hettinger ліёмқёмқҳ мҪ”л“ң кІҖнҶ м—җм„ңлҠ” BListк°Җ мқјл°ҳ listліҙлӢӨ м„ нҳёлҗ  л§Ңн•ң мӮ¬лЎҖлҘј м°ҫм§Җ лӘ»н–Ҳм§Җл§Ң, мқҙлҠ” BListк°Җ м—Ҷм—Ҳкё° л•Ңл¬ём—җ л°ңмғқн•ң нҺён–Ҙмқј мҲҳ мһҲлӢӨкі  мқём •н–ҲмҠөлӢҲлӢӨ. лӘҮ лӢ¬ нӣ„ comp.lang.pythonм—җ BList м„ұкіө мӮ¬лЎҖлҘј л¬ёмқҳн•  кі„нҡҚмқҙм—Ҳмңјл©°, л§Ңм•Ҫ к·ёлҹ¬н•ң мӮ¬лЎҖк°Җ мһҲлӢӨл©ҙ collections лӘЁл“Ҳ нҸ¬н•Ём—җ л¬ём ңк°Җ м—ҶлӢӨкі  л°қнҳ”мҠөлӢҲлӢӨ. BListлҠ” н•ҷмҠө кіЎм„ мқҙ кұ°мқҳ 0м—җ к°Җк№қкё° л•Ңл¬ём—җ, мң мқјн•ң лӢЁм җмқҖ нҠ№м • мһ‘м—…м—җ к°ҖмһҘ м Ғн•©н•ң лҚ°мқҙн„° кө¬мЎ°лҘј кІ°м •н•ҳлҠ” лҚ° мһҲм–ҙ нҳјлһҖмқ„ м•јкё°н•  мҲҳ мһҲлӢӨлҠ” м җмқҙлқјкі  м–ёкёүн–ҲмҠөлӢҲлӢӨ.

к°ңмҡ” (Abstract)

list м—°мӮ°мқҳ мқјл°ҳм Ғмқё кІҪмҡ°лҠ” мһ‘мқҖ лҰ¬мҠӨнҠём—җм„ң л°ңмғқн•©лӢҲлӢӨ. нҳ„мһ¬ л°°м—ҙ кё°л°ҳмқҳ list кө¬нҳ„мқҖ м°ёмЎ° м§Җм—ӯм„ұ(locality of reference)мқҙ к°•н•ҳкі  л©”лӘЁлҰ¬ н• лӢ№ мһ‘м—…мқҙ л№ҲлІҲн•ҳм§Җ м•Ҡкё° л•Ңл¬ём—җ мһ‘мқҖ лҰ¬мҠӨнҠём—җм„ң нғҒмӣ”н•ң м„ұлҠҘмқ„ ліҙмһ…лӢҲлӢӨ. к·ёлҹ¬лӮҳ л°°м—ҙмқҖ мҡ”мҶҢлҘј мӮҪмһ…н•ҳкі  мӮӯм ңн•ҳлҠ” лҚ° O(n) мӢңк°„мқҙ мҶҢмҡ”лҗ  мҲҳ мһҲмңјл©°, лҰ¬мҠӨнҠёк°Җ м»Өм§ҲмҲҳлЎқ л¬ём ңк°Җ лҗ  мҲҳ мһҲмҠөлӢҲлӢӨ.

мқҙ PEPлҠ” л°°м—ҙкіј нҠёлҰ¬м Ғмқё мёЎл©ҙмқ„ лӘЁл‘җ к°Җм§ҖлҠ” мғҲлЎңмҡҙ лҚ°мқҙн„° нғҖмһ…мқё BListлҘј мҶҢк°ңн•©лӢҲлӢӨ. BListлҠ” кё°мЎҙ л°°м—ҙ кё°л°ҳ кө¬нҳ„кіј лҸҷмқјн•ҳкІҢ мһ‘мқҖ лҰ¬мҠӨнҠём—җм„ң мҡ°мҲҳн•ң м„ұлҠҘмқ„ м ңкіөн•ҳм§Җл§Ң, лҢҖл¶Җл¶„мқҳ м—°мӮ°м—җм„ң лҚ” лӮҳмқҖ м җк·јм Ғ м„ұлҠҘ(asymptotic performance)мқ„ м ңкіөн•©лӢҲлӢӨ. мқҙ PEPлҠ” BList нғҖмһ…мқ„ Pythonм—җ нҸ¬н•Ён•ҳкё° мң„н•ң л‘җ к°Җм§Җ мғҒнҳё л°°нғҖм Ғмқё м ңм•Ҳмқ„ м ңмӢңн–ҲмҠөлӢҲлӢӨ:

  1. collections лӘЁл“Ҳм—җ м¶”к°Җ
  2. кё°мЎҙ list нғҖмһ…мқ„ лҢҖмІҙ

лҸҷкё° (Motivation)

BListлҠ” мһ‘мқҖ мһ…л Ҙм—җм„ңлҠ” мһҳ мһ‘лҸҷн•ҳм§Җл§Ң, л°°м—ҙ кё°л°ҳ лҰ¬мҠӨнҠёмқҳ кё°м Җ O(n) лҸҷмһ‘мңјлЎң мқён•ҙ нҒ° мһ…л Ҙм—җм„ңлҠ” O(n**2) мӢңк°„мқҙ мҶҢмҡ”лҗҳлҠ” м§ҒкҙҖм Ғмқё м•Ңкі лҰ¬мҰҳмқ„ лӢӨмӢң мһ‘м„ұн•ҙм•ј н•ҳлҠ” мўҢм Ҳк°җм—җм„ң л№„лЎҜлҗҳм—ҲмҠөлӢҲлӢӨ. Python 2.4м—җ лҸ„мһ…лҗң deque нғҖмһ…мқҖ л№ лҘё FIFO нҒҗ(queue)к°Җ н•„мҡ”н•ң к°ҖмһҘ мқјл°ҳм Ғмқё л¬ём ңлҘј н•ҙкІ°н–ҲмҠөлӢҲлӢӨ. к·ёлҹ¬лӮҳ deque нғҖмһ…мқҖ кёҙ лҰ¬мҠӨнҠё мӨ‘к°„м—җм„ң мҡ”мҶҢлҘј л°ҳліөм ҒмңјлЎң мӮҪмһ…н•ҳкұ°лӮҳ мӮӯм ңн•ҙм•ј н•  л•ҢлҠ” лҸ„мӣҖмқҙ лҗҳм§Җ м•ҠмҠөлӢҲлӢӨ.

лӢӨм–‘н•ң лҚ°мқҙн„° кө¬мЎ°к°Җ мӮҪмһ… л°Ҹ мӮӯм ңм—җ лҢҖн•ҙ мўӢмқҖ м җк·јм Ғ м„ұлҠҘмқ„ м ңкіөн•ҳм§Җл§Ң, лӢӨлҘё м—°мӮ°м—җм„ңлҠ” O(n) м„ұлҠҘмқ„ к°Җм§Җкұ°лӮҳ (мҳҲ: м—°кІ° лҰ¬мҠӨнҠё, Linked Lists), мһ‘мқҖ лҰ¬мҠӨнҠём—җм„ңлҠ” м„ұлҠҘмқҙ л–Ём–ҙм§‘лӢҲлӢӨ (мҳҲ: мқҙм§„ нҠёлҰ¬, Binary Trees л°Ҹ мҠӨнӮө лҰ¬мҠӨнҠё, Skip Lists).

мқҙ PEPм—җм„ң м ңм•Ҳн•ҳлҠ” BList нғҖмһ…мқҖ B+Treeмқҳ мӣҗлҰ¬лҘј кё°л°ҳмңјлЎң н•ҳл©°, л°°м—ҙкіј нҠёлҰ¬м Ғмқё мёЎл©ҙмқ„ лӘЁл‘җ к°Җм§‘лӢҲлӢӨ. BListлҠ” мһ‘мқҖ лҰ¬мҠӨнҠём—җм„ң л°°м—ҙкіј к°ҷмқҖ м„ұлҠҘмқ„ м ңкіөн•ҳлҠ” лҸҷмӢңм—җ, лӘЁл“  мӮҪмһ… л°Ҹ мӮӯм ң м—°мӮ°м—җ лҢҖн•ҙ O(log n)мқҳ м җк·јм Ғ м„ұлҠҘмқ„ м ңкіөн•©лӢҲлӢӨ. лҳҗн•ң BListлҠ” лӮҙл¶Җм ҒмңјлЎң Copy-on-WriteлҘј кө¬нҳ„н•ҳм—¬ getsliceмҷҖ к°ҷмқҖ м—°мӮ°лҸ„ O(log n) мӢңк°„мқҙ мҶҢмҡ”лҗ©лӢҲлӢӨ. лӢӨмқҢ н‘ңлҠ” нҳ„мһ¬ л°°м—ҙ кё°л°ҳ list кө¬нҳ„кіј BListмқҳ м җк·јм Ғ м„ұлҠҘмқ„ л№„көҗн•©лӢҲлӢӨ:

Pythonмқҳ л°°м—ҙ кё°л°ҳ listмҷҖ BListм—җ лҢҖн•ң кҙ‘лІ”мң„н•ң мӢӨмҰқм Ғ л№„көҗлҠ”м—җм„ң нҷ•мқён•  мҲҳ мһҲмҠөлӢҲлӢӨ.

мӮ¬мҡ© мӮ¬лЎҖ нҠёл Ҳмқҙл“ңмҳӨн”„ (Use Case Trade-offs)

BListлҠ” лӘЁл“  м—°мӮ°м—җм„ң мҡ°мҲҳн•ҳм§ҖлҠ” м•Ҡм§Җл§Ң, л§ҺмқҖ м—°мӮ°м—җм„ң мҡ°мӣ”н•ң м„ұлҠҘмқ„ м ңкіөн•©лӢҲлӢӨ. нҠ№м • мӮ¬мҡ© мӮ¬лЎҖм—җ л§һлҠ” мҳ¬л°”лҘё лҚ°мқҙн„° нғҖмһ…мқ„ м„ нғқн•ҳлҠ” кІғмқҖ м–ҙл–Ө м—°мӮ°мқҙ мЈјлЎң мӮ¬мҡ©лҗҳлҠ”м§Җм—җ л”°лқј лӢ¬лқјм§‘лӢҲлӢӨ. лӮҙмһҘ(built-in) лҚ°мқҙн„° нғҖмһ…мңјлЎң м„ нғқн•ҳлҠ” кІғмқҖ лӢӨм–‘н•ң мӮ¬мҡ© мӮ¬лЎҖмқҳ мӨ‘мҡ”м„ұкіј м„ұлҠҘ м°Ёмқҙмқҳ нҒ¬кё°лҘј к· нҳ• мһҲкІҢ кі л Өн•ҙм•ј н•©лӢҲлӢӨ.

мһ‘мқҖ лҰ¬мҠӨнҠёмқҳ мқјл°ҳм Ғмқё мӮ¬мҡ© мӮ¬лЎҖмқҳ кІҪмҡ°, л°°м—ҙ кё°л°ҳ listмҷҖ BListлҠ” мң мӮ¬н•ң м„ұлҠҘ нҠ№м„ұмқ„ к°Җм§‘лӢҲлӢӨ.

лҢҖк·ңлӘЁ лҰ¬мҠӨнҠёмқҳ лҚң мқјл°ҳм Ғмқё мӮ¬мҡ© мӮ¬лЎҖм—җм„ңлҠ” кё°мЎҙ л°°м—ҙ кё°л°ҳ listк°Җ BList м°ёмЎ° кө¬нҳ„ліҙлӢӨ лӣ°м–ҙлӮң л‘җ к°Җм§Җ мқјл°ҳм Ғмқё мӮ¬мҡ© мӮ¬лЎҖк°Җ мһҲмҠөлӢҲлӢӨ:

  • лҢҖк·ңлӘЁ LIFO мҠӨнғқ: .append() л°Ҹ .pop(-1) м—°мӮ°мқҙ л§ҺмқҖ кІҪмҡ°. к°Ғ м—°мӮ°мқҖ л°°м—ҙ кё°л°ҳ listмқҳ кІҪмҡ° O(1)мқҙм§Җл§Ң, BListмқҳ кІҪмҡ° O(log n)мһ…лӢҲлӢӨ.
  • нҒ¬кё°к°Җ ліҖкІҪлҗҳм§Җ м•ҠлҠ” лҢҖк·ңлӘЁ лҰ¬мҠӨнҠё: getitem л°Ҹ setitem нҳём¶ңмқҙ л°°м—ҙ кё°л°ҳ listмқҳ кІҪмҡ° O(1)мқҙм§Җл§Ң, BListмқҳ кІҪмҡ° O(log n)мһ…лӢҲлӢӨ.

10,000к°ң мҡ”мҶҢмқҳ лҰ¬мҠӨнҠём—җ лҢҖн•ң м„ұлҠҘ н…ҢмҠӨнҠём—җм„ң, BListлҠ” мқҙ л‘җ к°Җм§Җ мӮ¬мҡ© мӮ¬лЎҖм—җ лҢҖн•ҙ к°Ғк°Ғ 50% л°Ҹ 5%мқҳ мӢӨн–ү мӢңк°„ мҰқк°ҖлҘј ліҙмҳҖмҠөлӢҲлӢӨ.

LIFO мӮ¬мҡ© мӮ¬лЎҖмқҳ м„ұлҠҘмқҖ лЈЁнҠё л…ёл“ң лӮҙм—җ к°ҖмһҘ мҳӨлҘёмӘҪ лҰ¬н”„м—җ лҢҖн•ң нҸ¬мқён„°лҘј мәҗмӢұн•ҳм—¬ O(n) мӢңк°„мңјлЎң к°ңм„ лҗ  мҲҳ мһҲмҠөлӢҲлӢӨ. нҒ¬кё°к°Җ ліҖкІҪлҗҳм§Җ м•ҠлҠ” лҰ¬мҠӨнҠёмқҳ кІҪмҡ°, мҲңм°Ё м ‘к·јмқҳ мқјл°ҳм Ғмқё кІҪмҡ°лҸ„ лЈЁнҠё л…ёл“ңмқҳ мәҗмӢұмқ„ нҶөн•ҙ O(n) мӢңк°„мңјлЎң к°ңм„ лҗ  мҲҳ мһҲмҠөлӢҲлӢӨ. к·ёлҹ¬лӮҳ мқҙлҹ¬н•ң м ‘к·ј л°©мӢқмқҳ м„ұлҠҘмқҖ м•„м§Ғ мӢӨмҰқм ҒмңјлЎң н…ҢмҠӨнҠёлҗҳм§Җ м•Ҡм•ҳмҠөлӢҲлӢӨ.

л§ҺмқҖ м—°мӮ°мқҙ л°°м—ҙ кё°л°ҳ listм—җм„ң BListлЎң м „нҷҳн•  л•Ң м—„мІӯлӮң мҶҚлҸ„ н–ҘмғҒ(O(n)м—җм„ң O(log n))мқ„ ліҙмһ…лӢҲлӢӨ. 10,000к°ң мҡ”мҶҢ лҰ¬мҠӨнҠём—җ лҢҖн•ң м„ұлҠҘ н…ҢмҠӨнҠём—җм„ң, BListмқҳ getslice, setslice л°Ҹ FIFO мҠӨнғҖмқј мӮҪмһ…/мӮӯм ңмҷҖ к°ҷмқҖ м—°мӮ°мқҖ л°°м—ҙ кё°л°ҳ listм—җ н•„мҡ”н•ң мӢңк°„мқҳ 1%м—җ л¶Ҳкіјн–ҲмҠөлӢҲлӢӨ.

  • лЈЁнҠё л…ёл“ңлҠ” лҰ¬мҠӨнҠём—җ 2к°ң лҜёл§Ңмқҳ мҡ”мҶҢк°Җ нҸ¬н•Ёлҗң кІҪмҡ°к°Җ м•„лӢҲл©ҙ мөңмҶҢ 2к°ңмқҳ мһҗмӢқмқ„ к°Җм§‘лӢҲлӢӨ.

  • л°ҳлҢҖ: мӢӨм ң м• н”ҢлҰ¬мјҖмқҙм…ҳм—җм„ңмқҳ мң мҡ©м„ұкіј м„ұлҠҘмқҖ мһ…мҰқлҗҳм§Җ м•Ҡм•ҳмҠөлӢҲлӢӨ.
