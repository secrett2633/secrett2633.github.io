<h1 id="pep-790-python-315-릴리스-일정"><a href="#pep-790-python-315-릴리스-일정">PEP 790 – Python 3.15 릴리스 일정</a></h1>
<ul>
<li><strong>작성자:</strong> Hugo van Kemenade</li>
<li><strong>상태:</strong> Active (활성)</li>
<li><strong>유형:</strong> Informational (정보성)</li>
<li><strong>주제:</strong> Release (릴리스)</li>
<li><strong>생성일:</strong> 2025년 4월 26일</li>
<li><strong>Python 버전:</strong> 3.15</li>
</ul>
<h2 id="개요-abstract"><a href="#개요-abstract">개요 (Abstract)</a></h2>
<p>이 문서는 Python 3.15 버전의 개발 및 릴리스 일정을 설명합니다.</p>
<h2 id="릴리스-관리자-및-담당-팀-release-manager-and-crew"><a href="#릴리스-관리자-및-담당-팀-release-manager-and-crew">릴리스 관리자 및 담당 팀 (Release Manager and Crew)</a></h2>
<ul>
<li><strong>3.15 릴리스 관리자:</strong> Hugo van Kemenade</li>
<li><strong>Windows 인스톨러:</strong> Steve Dower</li>
<li><strong>Mac 인스톨러:</strong> Ned Deily</li>
<li><strong>문서화:</strong> Julien Palard</li>
</ul>
<h2 id="릴리스-일정-release-schedule"><a href="#릴리스-일정-release-schedule">릴리스 일정 (Release Schedule)</a></h2>
<h3 id="3150-일정"><a href="#3150-일정">3.15.0 일정</a></h3>
<p>아래 날짜들은 PEP 602에 정의된 대로 기능 버전(feature versions) 간 12개월의 릴리스 주기를 따르기 위해 17개월의 개발 기간을 사용합니다.</p>
<p><strong>실제 (Actual):</strong></p>
<ul>
<li><strong>3.15 개발 시작:</strong> 2025년 5월 7일 (수요일)</li>
</ul>
<p><strong>예상 (Expected):</strong></p>
<ul>
<li><strong>3.15.0 alpha 1:</strong> 2025년 10월 14일 (화요일)</li>
<li><strong>3.15.0 alpha 2:</strong> 2025년 11월 18일 (화요일)</li>
<li><strong>3.15.0 alpha 3:</strong> 2025년 12월 16일 (화요일)</li>
<li><strong>3.15.0 alpha 4:</strong> 2026년 1월 13일 (화요일)</li>
<li><strong>3.15.0 alpha 5:</strong> 2026년 2월 10일 (화요일)</li>
<li><strong>3.15.0 alpha 6:</strong> 2026년 3월 10일 (화요일)</li>
<li><strong>3.15.0 alpha 7:</strong> 2026년 4월 7일 (화요일)</li>
<li><strong>3.15.0 beta 1:</strong> 2026년 5월 5일 (화요일) (이 시점 이후로는 새로운 기능이 추가되지 않습니다.)</li>
<li><strong>3.15.0 beta 2:</strong> 2026년 5월 26일 (화요일)</li>
<li><strong>3.15.0 beta 3:</strong> 2026년 6월 16일 (화요일)</li>
<li><strong>3.15.0 beta 4:</strong> 2026년 7월 14일 (화요일)</li>
<li><strong>3.15.0 release candidate 1 (RC 1):</strong> 2026년 7월 28일 (화요일)</li>
<li><strong>3.15.0 release candidate 2 (RC 2):</strong> 2026년 9월 1일 (화요일)</li>
<li><strong>3.15.0 final:</strong> 2026년 10월 1일 (목요일)</li>
</ul>
<p>이후 버그픽스(bugfix) 릴리스는 약 2개월마다 진행될 예정입니다.</p>
<h3 id="315-수명-주기-315-lifespan"><a href="#315-수명-주기-315-lifespan">3.15 수명 주기 (3.15 Lifespan)</a></h3>
<p>Python 3.15는 약 2년 동안 매 두 번째 달마다 버그픽스 업데이트를 받을 예정입니다. 3.17.0 final 버전이 릴리스될 시점에 3.15의 최종 버그픽스 업데이트가 릴리스될 것입니다. 그 후에는 3.15.0 final 릴리스 이후 5년까지, 즉 대략 2031년 10월까지 3년 동안 보안 업데이트(소스 코드만 제공)가 릴리스될 것으로 예상됩니다.</p>
