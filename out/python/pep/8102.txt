<p><strong>원문 링크:</strong> <a href="https://peps.python.org/pep-8102/" target="_blank" rel="noopener noreferrer">PEP 8102 - 2021 Term Steering Council election</a></p>

<h1 id="pep-8102-2021년-임기-스티어링-카운슬-선거-2021-term-steering-council-election"><a href="#pep-8102-2021년-임기-스티어링-카운슬-선거-2021-term-steering-council-election">PEP 8102 – 2021년 임기 스티어링 카운슬 선거 (2021 Term Steering Council election)</a></h1>
<h2 id="개요"><a href="#개요">개요</a></h2>
<p>이 문서는 PEP 13에 명시된 바와 같이, 2020년 12월에 실시된 Python 스티어링 카운슬 (Steering Council) 선거의 일정 및 세부 사항을 설명합니다. 이 선거는 2021년 임기를 위한 스티어링 카운슬 선거였습니다.</p>
<p><strong>작성자:</strong> Ewa Jodlowska, Ee Durbin, Joe Carey
<strong>후원자:</strong> Brett Cannon
<strong>상태:</strong> 최종 (Final)
<strong>유형:</strong> 정보 (Informational)
<strong>주제:</strong> 거버넌스 (Governance)
<strong>생성일:</strong> 2020년 10월 29일</p>
<h2 id="선거-관리-election-administration"><a href="#선거-관리-election-administration">선거 관리 (Election Administration)</a></h2>
<p>스티어링 카운슬은 Python Software Foundation (PSF)의 인프라 담당 이사(Director of Infrastructure) Ee Durbin과 회계 관리자(Accounting Manager) Joe Carey를 선거 공동 관리자로 임명했습니다.</p>
<p>PSF 전무이사(Executive Director) Ewa Jodlowska는 선거와 관련된 공지사항을 전달했습니다.</p>
<h2 id="일정-schedule"><a href="#일정-schedule">일정 (Schedule)</a></h2>
<p>선거는 2주간의 후보 지명 기간과 2주간의 투표 기간으로 구성되었습니다.</p>
<ul>
<li><strong>후보 지명 기간:</strong> 2020년 11월 1일부터 2020년 11월 16일 12:00 UTC (Anywhere on Earth 기준 2020년 11월 15일 말)까지였습니다. 이 내용은 python-committers에 공지되었고 discuss.python.org에서 진행되었습니다.</li>
<li><strong>투표 기간:</strong> 2020년 12월 1일 12:00 UTC부터 2020년 12월 16일 12:00 UTC (Anywhere on Earth 기준 2020년 12월 15일 말)까지였습니다.</li>
</ul>
<h2 id="후보자-candidates"><a href="#후보자-candidates">후보자 (Candidates)</a></h2>
<p>후보자는 핵심 팀 구성원(core team member)에 의해 지명되어야 합니다. 후보자 본인이 핵심 팀 구성원인 경우, 스스로 지명할 수 있었습니다.</p>
<p><strong>지명된 후보자 (알파벳순):</strong></p>
<ul>
<li>Barry Warsaw</li>
<li>Brett Cannon</li>
<li>Carol Willing</li>
<li>Christian Heimes</li>
<li>David Mertz</li>
<li>Matt Harrison</li>
<li>Mariatta</li>
<li>Pablo Galindo Salgado</li>
<li>Raymond Hettinger</li>
<li>Thomas Wouters</li>
</ul>
<p><strong>철회된 지명:</strong> 없음</p>
<h2 id="유권자-명부-voter-roll"><a href="#유권자-명부-voter-roll">유권자 명부 (Voter Roll)</a></h2>
<p>모든 활동적인 Python 핵심 팀 구성원 (active Python core team members)은 투표 자격이 있었습니다. 활동 상태는 PEP 13에 설명된 바와 같이 <code>python/voters</code> 소프트웨어를 통해 결정되었습니다.</p>
<p>투표 용지는 이 선거를 위한 Python 유권자 명부를 기반으로 배포되었습니다. 이 파일에는 개인 이메일 주소가 포함되어 있어 공개되지 않았지만, 유권자 명부가 생성될 때 이름으로 된 전체 유권자 명부는 제공될 예정이었습니다.</p>
<h2 id="선거-구현-election-implementation"><a href="#선거-구현-election-implementation">선거 구현 (Election Implementation)</a></h2>
<p>선거는 Helios Voting Service를 사용하여 진행되었습니다.</p>
<p><strong>설정 (Configuration):</strong></p>
<ul>
<li><strong>짧은 이름 (Short name):</strong> <code>2021-python-steering-council</code></li>
<li><strong>이름 (Name):</strong> <code>2021 Python Steering Council Election</code></li>
<li><strong>설명 (Description):</strong> PEP 13에 명시된 Python 스티어링 카운슬 선거입니다. 이는 2021년 임기를 위한 스티어링 카운슬 선거입니다.</li>
<li><strong>유형 (type):</strong> <code>Election</code></li>
<li><strong>유권자 별칭 사용 (Use voter aliases):</strong> [X] (활성화됨)</li>
<li><strong>답변 순서 무작위화 (Randomize answer order):</strong> [X] (활성화됨)</li>
<li><strong>비공개 (Private):</strong> [X] (활성화됨)</li>
<li><strong>도움말 이메일 주소 (Help Email Address):</strong> <code>psf-election@python.org</code></li>
<li><strong>투표 시작 시간 (Voting starts at):</strong> 2020년 12월 1일 00:00 UTC</li>
<li><strong>투표 종료 시간 (Voting ends at):</strong> 2020년 12월 16일 12:00 UTC</li>
</ul>
<p>이러한 설정은 다음과 같은 선거를 만듭니다.</p>
<ul>
<li>투표는 일반에 공개되지 않으며, 유권자 명부에 있는 사람만 참여할</li>
<li>수 있습니다.</li>
<li>투표가 시작될 때 투표 용지가 이메일로 발송됩니다.</li>
<li>후보자는 편향을 피하기 위해 무작위 순서로 제시됩니다.</li>
<li>유권자의 신원과 투표 용지는 암호화 기술 발전으로부터 보호됩니다.</li>
</ul>
<p><strong>질문 (Questions):</strong></p>
<ul>
<li><strong>질문 1:</strong> 0개에서 (승인 투표) 답변 중 선택하세요.
<ul>
<li><strong>결과 유형 (Result Type):</strong> 절대 (absolute)</li>
<li><strong>질문:</strong> Python 스티어링 카운슬 후보자를 선택하세요.</li>
<li><strong>답변 #1 - #N:</strong> "후보자 (Candidates)" 섹션의 후보자들</li>
</ul>
</li>
</ul>
