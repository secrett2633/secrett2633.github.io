<h2>PEP 8105 – 2024년 임기 Steering Council 선거</h2>
<p><strong>작성자:</strong> Ee Durbin &#x3C;>
<strong>후원자:</strong> Thomas Wouters &#x3C;>
<strong>상태:</strong> 최종 (Final)
<strong>유형:</strong> 정보 제공 (Informational)
<strong>주제:</strong> 거버넌스 (Governance)
<strong>생성일:</strong> 2023년 10월 23일</p>
<hr>
<h3>초록 (Abstract)</h3>
<p>이 문서는 PEP 13에 명시된 바와 같이 Python Steering Council의 2023년 선거(2024년 임기, 즉 Python 3.13) 일정 및 기타 세부 사항을 설명합니다.</p>
<h3>선거 관리 (Election Administration)</h3>
<p>Steering Council은 Python Software Foundation의 인프라 책임자인 Ee Durbin을 선거 관리자로 임명했습니다.</p>
<h3>일정 (Schedule)</h3>
<p>후보 지명 기간은 2주였으며, 이어서 2주간의 투표 기간이 있었습니다.</p>
<ul>
<li><strong>후보 지명 기간:</strong> 2023년 11월 9일 ~ 2023년 11월 22일 AoE (Anywhere on Earth)</li>
<li><strong>투표 기간:</strong> 2023년 11월 27일 ~ 2023년 12월 11일 AoE</li>
</ul>
<h3>후보자 (Candidates)</h3>
<p>후보자는 핵심 팀원(core team member)의 지명을 받아야 합니다. 후보자가 핵심 팀원인 경우, 스스로를 지명할 수 있습니다.</p>
<p><strong>지명된 후보자 (이름 가나다순):</strong></p>
<ul>
<li>Barry Warsaw</li>

<p><strong>철회된 지명:</strong> 없음</p>
<h3>유권자 명부 (Voter Roll)</h3>
<p>모든 활동 중인 Python 핵심 팀원(active Python core team members)은 투표 자격이 있습니다. 활동 상태는 PEP 13에 설명된 대로 결정되며, python/voters 소프트웨어를 통해 구현됩니다.</p>
<p>투표 용지는 이번 선거의 Python Voter Roll을 기반으로 배포되었습니다. 이 파일은 개인 이메일 주소를 포함하고 있어 공개되지 않지만, 모든 유권자 이름 목록이 포함된 전체 유권자 명부(Complete Voter Roll)는 확인할 수 있습니다.</p>
