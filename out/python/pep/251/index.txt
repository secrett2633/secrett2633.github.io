<h2>PEP 251 – Python 2.2 릴리스 일정</h2>
<p>이 문서는 Python 2.2의 개발 및 릴리스 일정을 설명하며, 주로 PEP 규모의 항목들에 초점을 맞춥니다. 작은 버그 수정 및 변경 사항은 첫 번째 베타 릴리스 전까지 발생했습니다. 아래 일정은 Python 2.2의 실제 릴리스 날짜를 나타냅니다. 이후의 Python 2.2 유지보수 릴리스는 별도의 PEP 문서에서 다룹니다.</p>
<h3>문서 개요</h3>
<ul>
<li><strong>작성자:</strong> Barry Warsaw, Guido van Rossum</li>
<li><strong>상태:</strong> Final (최종)</li>
<li><strong>유형:</strong> Informational (정보성)</li>
<li><strong>주제:</strong> Release (릴리스)</li>
<li><strong>생성일:</strong> 2001년 4월 17일</li>
<li><strong>Python 버전:</strong> 2.2</li>
<li><strong>이력:</strong> 2001년 8월 14일</li>
</ul>
<h3>릴리스 일정</h3>
<p>다음은 Python 2.2의 최종 릴리스 날짜입니다.</p>
<ul>
<li><strong>2001년 12월 21일:</strong> 2.2 [릴리스 완료] (최종 릴리스)</li>
<li><strong>2001년 12월 14일:</strong> 2.2c1 [릴리스 완료]</li>
<li><strong>2001년 11월 14일:</strong> 2.2b2 [릴리스 완료]</li>
<li><strong>2001년 10월 19일:</strong> 2.2b1 [릴리스 완료]</li>
<li><strong>2001년 9월 28일:</strong> 2.2a4 [릴리스 완료]</li>
<li><strong>2001년 9월 7일:</strong> 2.2a3 [릴리스 완료]</li>
<li><strong>2001년 8월 22일:</strong> 2.2a2 [릴리스 완료]</li>
<li><strong>2001년 7월 18일:</strong> 2.2a1 [릴리스 완료]</li>
</ul>
<h3>릴리스 관리자 (Release Manager)</h3>
<p>Python 2.2의 릴리스 관리자는 Barry Warsaw였습니다.</p>
<h3>릴리스 방식 (Release Mechanics)</h3>
<p>Python 2.2 릴리스에서는 새로운 방식이 시도되었습니다. 모든 알파(alpha), 베타(beta) 또는 기타 릴리스가 나오기 일주일 전에, 릴리스가 될 브랜치(branch)를 분기(fork)했습니다. 이 브랜치에 대한 변경 사항은 릴리스 관리자와 그가 지정한 '봇(bots)'으로 제한되었습니다. 이 실험은 성공적이었으며 향후 릴리스에서도 적용되어야 한다고 평가되었습니다. 실제 릴리스 방식에 대한 자세한 내용은 PEP 101을 참조하십시오.</p>
<h3>Python 2.2의 새로운 기능</h3>
<p>Python 2.2에 도입된 새로운 기능들은 다음과 같습니다. 더 자세한 내용은 Python 배포판의 <code>Misc/NEWS</code> 파일 또는 Andrew Kuchling의 "What's New in Python 2.2" 문서를 참조하십시오.</p>
<ul>
<li><strong>Iterators (이터레이터)</strong> (PEP 234)</li>
<li><strong>Generators (제너레이터)</strong> (PEP 255)</li>
<li><strong>long ints와 plain ints의 통합</strong> (PEP 237)</li>
<li><strong>나눗셈 연산자 변경</strong> (PEP 238)</li>
<li><strong>타입(types)과 클래스(classes)의 통합</strong> (PEP 252, PEP 253)</li>
</ul>
